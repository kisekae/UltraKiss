Kisekae UltraKiss Installation

Kisekae UltraKiss is distributed as open source software from GitHub under the project Kisekae UltraKiss

The UltraKiss product has been developed as a platform independent product for Microsoft, Unix and Apple computer systems.

UltraKiss requires a Java run time environment.  Kisekae UltraKiss is supplied with a JRE version 8 system.

 

Effective with UltraKiss release 3.3b, Kisekae UltraKiss is distributed as a fully bundled application for Microsoft Windows systems.

No separate installation of Java software is required.

 

Linux and Apple systems can run UltraKiss directly from the kisekae.jar file using the supplied JRE version 8 system.
