Editing the CNF

Did you know that you can edit your KiSS configuration CNF file within UltraKiss and reload your KiSS set to see the results immediately?

Press F11 or use View-Active Configuration to invoke the Text Editor tool.  Make your changes and when you exit the editor your KiSS set can be reloaded.

This feature makes it easy to test configuration code changes. Your configuration file is not permanently updated until you save the file in the Text Editor or save the KiSS set with the File-Save command.
