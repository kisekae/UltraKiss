GIF to CEL Images

Did you know that you can easily convert GIF, JPG, BMP, PNG, and PPM images to CELs?
Load an image into the Image Editor or Color Editor tool.  Use File-Save As and specify a CEL file name extension.

UltraKiss will generate a KCF file for all palette type cels.  The KCF file will have the same name as the image file. Images will be converted to Cherry Kiss cels if they contain more than 255 colors.

Furthermore, if PlayFKiss or other viewer compatibility options are set, all imported images are automatically converted to CEL format.
