Icon Images

Did you know that you can replace or change the UltraKiss icon? You can even have animated icons!

UltraKiss icon images are GIF files stored in the 'Icons ' directory in your installation file folder. The files are named 'icon1', 'icon2', 'icon3', ... and so on.

You can select the icon that UltraKiss uses from the Icon display in the Options-UKiss section.

You can replace these images, add new images, or delete them as you wish.
