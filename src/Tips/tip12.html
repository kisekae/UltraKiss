Selecting Objects

Did you know that you can visually edit your KiSS sets?

To enable edit selections, the edit toggle button on the toolbar must be selected.

When edits are enabled objects underneath the mouse pointer can be selected by performing an Alt-leftclick or by right clicking on the object and selecting the object using the pop-up context menu.

A selected object will have a blue marquee box drawn around it.

You can also select all objects within a selection area by pressing the Alt key and dragging. You can even select multiple objects by pointing at each object and performing an Alt-Ctrl-leftclick.

Objects can be unselected with an Alt-rightclick or by drawing a selection area with the right mouse button.

Selected objects can be copied, pasted, cut from the page, or have their properties changed or be modified without causing FKiSS events to be fired.
