Importing Images

Did you know that you can import new images into your KiSS set, directly from another LZH or ZIP file?

Use Edit-Import Image-To New Object.  Select an LZH or ZIP archive file.  Choose a CEL image or other image from the pop-up selection dialog. Choose the appropriate KCF file if prompted.

The CEL and KCF are added to your CNF file.

You can also import BMP, GIF, JPG, PNG, PPM, PBM, and PGM images stored in a file directory directly into your KiSS set and have them converted to CEL images when the set is saved. To save images in a different format, see the Options-Export option settings.
