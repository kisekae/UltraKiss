Copying Objects

Selected objects can be copied to the clipboard.  Use the Edit-Copy command or the Ctrl-C key.

When objects are copied to the clipboard they can be pasted into the same page or new pages as many times as desired.

Copied objects, when pasted into a new page, automatically update the KiSS configuration file so that the cel images are associated with the new page. Copied objects pasted into the same page create duplicate ambiguous cels for each cel image in the object.
