Indexing KiSS sets

Did you know that you can construct an index to access all your KiSS sets on your computer?
The File-Search command can be used to automatically construct an HTML index page with hyperlinks to KiSS files on your computer. Simply invoke the Search module and choose to scan a file directory on your system.  This will construct an index into your KiSS sets. You can then use the UltraKiss Portal to easily access the index page to view all your sets. 

You can also use the File-Search command to index KiSS sets found on remote web sites.  Simply specify a URL to a web site page and the Search module will scan the remote site and build hyperlinks to any KiSS sets stored on the site.

Use the UltraKiss Portal to load and view any indexed KiSS set.
