Pasting Objects

Objects in the clipboard can be pasted into the same page or new pages.  Use the Edit-Paste command or the Ctrl-V key.

When objects are pasted into the same page as duplicate objects, new object numbers are created in the KiSS configuration file. Duplicate ambiguous cel references are created for each cel image in the object.

When objects are pasted into different pages cel references for the existing object are marked as being visible on the new page.
