The UltraKiss Portal

Did you know that UltraKiss can download and run KiSS sets from the Internet?

UltraKiss has a built-in web browser, called the Portal, that can be used to directly load and open KiSS sets from the Internet or stored on your local computer   You start the Portal with the File-Open Portal command, or press the Kanji icon on the toolbar. 

The portal is like a web browser. It displays web pages with hyperlinks to other web pages or KiSS sets. You can set the Portal home page in the Options-UKiss section. Portal pages should adhere to HTML 3.2 specifications.

If you click on any link to an LZH or ZIP file, UltraKiss will download and run the file if it contains a KiSS configuration element.
