Copying Palettes

Have you ever wanted to create a new palette group within a KCF palette file?
With the Color Editor, selected palette colors can be copied and pasted. Use the Edit-Copy command or the Ctrl-C key to copy selected colors.

Use Edit-Insert Palette Group or Edit-Add Palette Group to create a new palette group.

Use Edit-Select Copy or otherwise manually select a set of colors in the new palette.

Use Edit-Paste to copy the selected colors to the new selection set.

If the copy selection set and paste selection set differ in size you will see a warning dialog, but you can still complete the transfer.
