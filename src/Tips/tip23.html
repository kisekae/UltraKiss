Converting to LZH

Did you know that you can easily package all your KiSS files together into an LZH file?

The File-Save As Archive command can be used to build your set into an  LZH or ZIP archive.

You must have loaded your CNF from a file directory that contains the CNF and other files.

The complete KiSS set will be written using the Archive Manager.  You can then add other files such as documentation with the Archive Manager tool.
