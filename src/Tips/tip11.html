Expansion Sets

Some KiSS sets are expansion sets. These sets are meant to be added to base sets to extend or replace the set capabilities.

To load an expansion set, first load the base set with File-Open. Then use File-Add to load the expansion set.

The expansion set can add new images and new CNF files to the original set. UltraKiss also supports the use of INCLUDE directives in the KiSS configuration file to automatically reference the required expansion sets.
