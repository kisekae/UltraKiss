Color Tinting

Have you ever wanted to change the color of an object?
1. Right click when your mouse is over an object and choose Image Context - Edit Palette.

2. In the Color Editor, use the eyedropper tool to select a color in the object.

3. Next, use the HSB slider to set the color selection sensitivity.

4. Finally, choose a new color using the Color Chooser selection tool.

If the preview image does not look right, Edit-Undo your change and play with the HSB slider to try again.

Exit the Color Editor and apply your change to the KiSS set.
