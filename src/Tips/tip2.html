Memory Configuration

UltraKiss is a Java application that is bound to the maximum memory pool size assigned to your Java Virtual Machine.

The default pool size is half half of the physical memory up to a physical memory size of 8 gigabytes (GB) on a 64 bit architecture machine. This is sufficient for all KiSS sets. The maximum memory pool size allocated is reported in the UltraKiss log file. To request a larger memory buffer pool you must modify the Java configuration. Failure to do so can result in poor performance or Out of Memory faults. 

If you start Java manually, use: 

java -Xmx200M -jar kisekae.jar
