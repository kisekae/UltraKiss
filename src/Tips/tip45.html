Cherry Kiss to Palette

Have you ever wanted to convert a Cherry Kiss cel to a palette cel?

Cherry Kiss images contain more than 255 colors.  These images do not have a palette.

Load the image in the Color Editor.  Use the Edit-Set Color Count command and enter a number less than 256.

Save the file with File-Save or File-Save As.

The converted image will be written as a palette cel with a KCF file having the same name as the CEL file.
