Securing your Set

To reduce intellectual property theft you can set the UltraKiss security feature.

In the Options-Edit section set the Enable Security option.

Secure sets cannot be edited in UltraKiss and the CNF information cannot be directly viewed.  This means that within UltraKiss, secure sets cannot be examined. However, other viewers or tools may still have access to your set information. Therefore, on a future release the LZH file may be encrypted to further protect your work from tools external to UltraKiss.

You must save your set to implement this feature as a set specific option.
