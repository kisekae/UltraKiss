Object Properties

Did you know that you can examine and change many properties of KiSS objects?
Right-click when your mouse is over any image in a KiSS set.  Select the Object Properties or Image Properties item from the contextual pop-up menu. Note that to enable edit selections, you must press the edit toggle button on the toolbar

Or, right-click when your mouse is over a page set or color set toolbar button to see the page or palette properties.

Or, choose the File-Properties menu item to see the properties of the current open item.

Or, choose View-Active Objects or press F12 to display an index of all KiSS objects in the set.
