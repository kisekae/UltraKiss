Splash Images

Did you know that you can change the background image on the screen?

Double click the image on the UltraKiss idle screen to see a different picture.

Background images are stored in the 'Splash ' directory in your installation file folder. The images are named 'back1', 'back2', 'back3', ... and so on. You can use JPG, GIF, or PNG files.

The splash image directory and the image prefix name is set in Options-UKiss section.

A random image is chosen when UltraKiss starts. You can replace or otherwise customize the images on the UltraKiss idle screen.
Further, if you store a text file in the Splash directory of the name "back_reference.txt" where each line of the text file begins with the image name 'back1', back2', 'back3', ... and so on, the remainder of the line after the image name can specify a description of the image or other such copyright attribution. This text is displayed at the bottom of the main UltraKiss screen window.
