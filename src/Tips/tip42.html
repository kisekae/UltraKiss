Selecting Palette Colors

Did you know that you can select one or more colors in a palette?
In the Color Editor, click any color in the Palette Panel.

This becomes the 'active color' and it shows on the toolbar.

Ctrl-click on any other color to add this color to the selection set.  This new color now becomes the active color.

A Shift-click will select the complete sequence of colors in palette index order up to the new selected color.

A Ctrl-rightclick removes the color from the selection set.
