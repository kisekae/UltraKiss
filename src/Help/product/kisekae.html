What is UltraKiss?

 

UltraKiss is a computer program that implements the Kisekae Set system, KiSS, a Japanese graphics system originally developed to facilitate costume changes on virtual dolls.  

 

What does UltraKiss do?

The original KiSS model was designed to create computer assisted costume changes for dolls or animation characters.  Kisekae sets are drawings of anime characters with many different costumes, packaged in a file for viewing the work.. KiSS is a presentation method that allows the artist to program certain actions as you interact with the anime doll.  

UltraKiss was developed to help artists build their KiSS sets. UltraKiss can be used to view KiSS dolls and help with the construction of KiSS sets. There is an integrated color editor tool to change the colors of images.  An image editor tool lets you change the size, shape, or structure of your images.  There is also an integrated audio and video player so that you can include different sounds or movies in your set. For packaging the work, UltraKiss has an LHA, ZIP, and JAR archive file manager that lets you convert between archive types and save your KiSS set in standard files or compressed archive files. 

UltraKiss has been designed to provide improved developer support so that it is easy to create new KiSS applications. 

 

How does UltraKiss work?

UltraKiss reads and interprets KiSS data files.  These files contain computer graphics images, color palettes and data control files.  Objects, such as the doll figure, costumes, and other items are constructed from layered image components that usually show only parts of the complete image.  Objects can be manipulated through user actions or through  a simple event processing model that enables timed animations or other actions if objects are touched or dragged across the screen.  Color variations, enabled through palette changes in UltraKiss, can also be used to produce more realistic results.  UltraKiss supports multimedia sound, graphic animation and worldwide web access to KiSS sets through an integrated Internet browser.  

 

 

Why is KiSS fun?

Image manipulation, in a traditional sense, has been a source of entertainment over the years.  It has always been fun to enhance or alter our perception of what we see.  Children explore many learning experiences while playing with paper dolls.  Others can have hours of fun solving more challenging puzzles and playing interesting games of skill.  

UltraKiss can also be used to create new computer applications for general entertainment or other purposes. You can build interactive presentations with active content, access web information, and deploy fully capable computer systems that process and manipulate image information.  UltraKiss fully supports the KiSS model and provides new features that extend the Kisekae Set system into domains beyond Japanese animation. 

 

Where does the name Kisekae come from?

Kisekae is an implementation of the Japanese Kisekae Set System.  Kisekae takes its name from the Japanese term kisekae ningyou, which refers to playing dress-up with dolls.

 

Where do I find KiSS material?

Artists from around the world create KiSS data sets from popular comic book and cartoon characters.  Most KiSS data sets are available on the world wide web.  Kisekae is a system that can grow with you as your interests and skills change.
