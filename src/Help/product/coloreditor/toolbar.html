The Color Editor Toolbar

The Color Editor Toolbar is an optional menu at the top of the screen that displays icons for common menu functions.  The toolbar icons and their functions are shown below. 

 

    New palette. New palettes are created from the New menu on the Color Editor main program screen.  16 color or 256 color palettes can be created.  New palettes are initialized to a standard set of colors.  16 color palettes are initialized to 8 variants of grey and 8 standard colors.  256 color palettes are initialized to the default colors of a 16 color palette, followed by 240 new colors that span the color circle.

    Open file. The Open command is used to open an existing KiSS palette file or any other type of image file recognized by the Kisekae UltraKiss program. 

    Save file. The Save command will save a palette or image file to the file directory or compressed archive file from which it was loaded.  For example, if a palette file or image cel was loaded from an archive file then any updated elements will be written to this archive file.  If the files were loaded from a file directory then the Save operation writes all updated elements into this directory.  

    Cut selection. The Cut operation will remove selected colors from the current palette.  Colors, when cut, are placed in the clipboard and can be referenced through a subsequent Edit-Paste command.  Cut colors are replaced with a black color.  

    Copy selection. The Copy operation copies selected colors to the clipboard.  The colors in the clipboard can be referenced through a subsequent Edit-Paste command.  Copied colors retain their order in the palette but are not associated with a fixed color index.  If colors are pasted into a palette they will be placed into the current palette in the order of the current color selection set.

    Paste selection. The Paste operation transfers colors from the clipboard into the current palette.  Colors, when pasted, are inserted into the currently selected color set of the palette.  If the current selection set does not contain the same number of colors as exist in the clipboard, then a warning message will be displayed.  If the paste operation is accepted the clipboard colors will be copied in order into the selection set.   

    Undo edit. The Color Editor supports full undo and redo edit recovery.  Most editing changes made to a KiSS palette file are reversible.  Undo support is available for Edit Cut, Copy, and Paste operations.  Transparency and background color changes can be undone.  Palette group additions and deletions can be undone.  Color selections can be undone.  Every undo operation reverts the palette to its prior state.

    Redo edit. The Color Editor supports full undo and redo edit recovery.  Most editing changes made to a KiSS palette file are reversible.  Redo support is available for Edit Cut, Copy, and Paste operations.  Transparency and background color changes can be redone.  Palette group additions and deletions can be redone.  Color selections can be redone.  Every redo operation applies the previously undone edit to restore the palette to its former state.

 

Color Control

   Hue, Saturation, and Brightness color selection.  Click on the little up-down arrows in the corner of the buttons to increase or decrease the selection sensitivity.  Hue sensitivity is a degree range from 0 to 360.  Saturation and Brightness sensitivity are percentage ranges from 0 to 100.

   Red, Green, and Blue color selection.  Click on the little up-down arrows in the corner of the buttons to increase or decrease the selection sensitivity.  Red, Green, and Blue sensitivity are percentage ranges from 0 to 100. 

 

Image Preview

    Region of Interest selection.  Left click and drag the mouse within the image preview pane to select a portion of the preview image.  The selected area will be expanded to fill the preview window.  Right click on the image to restore it to its original size.

    Magnify.  Left click on the preview image to magnify it.  Right click to reduce the magnification.

    Image Positioning.  Left click and drag to move a magnified image within the preview pane.  Image movement is only possible if the image is magnified and exceeds the preview pane size.

    Color Pickup.  Click on the preview image to select the pixel color with the eyedropper.  The selected color will become the Color Editor active color.
