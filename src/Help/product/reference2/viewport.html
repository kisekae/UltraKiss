viewport() 

Syntax:     viewport (X-offset, Y-offset)

A viewport() command is used to move the window that has been set to constrain the display the KiSS set. The window size is defined throug a windowsize() command and this creates a window which is smaller than the playfield size set the the KiSS configuration file.

The viewport X-offset andY-offset are integer parameters that specify relative horizontal and vertical movement offsets, in pixels, for the current window. Negative values move the window to the left and up. Positive values move the window to the right and down.

Attempts to move the window beyond the playfield boundaries will fail silently.

 

Related actions: windowsize()

 

Compatibility:

FKiSS 1 allows literal names or objects. viewport (-10,0) Moves the window 10 pixels to the left
FKiSS 3 allows variables for objects.   let(X,10) viewport (X,0)  Moves the window 10 pixels to the right

 

 Example: 

;@windowsize(-100,-100)    ; Make a window 100 pixels smaller in width and height
;@viewport(-10,22)         ; Move the window 10 pixels left and 22 pixels down
