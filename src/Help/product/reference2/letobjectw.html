letobjectw()

Syntax:     letobjectw (variable,target)

The letobjectw() statement returns the current width of a cel or object on the currently visible page.

The same object on different page can have a different width.

The target can be a cel name or an object group specifier. FKiSS 3 allowed only object groups for the letobjectw() statement, FKiSS 5 allows cel names.

 

Related actions: letobjecth() letobjectx() letobjecty()

 

 

Compatibility:

FKiSS 5   letobjectw(N,"shirt.cel")  Variable N is set to the current width of "shirt.cel".

 

Note: The UltraKiss letobjectw() command is viewer specific and not compatible with other viewers.

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letobjectw(N,"shirt.cel") ; N contains the current width for the specified cel
;@letobjectw(N,#3) ; N contains the current width for object #3

 

 Example: 

;[Cel Section]
#4 rightshoe.gif : 0 ;
#4 leftshoe.gif : 0 ;
#4 sailormoon.gif : 0 ;
 
;[FKiSS Section]
;@EventHandler
 
;@press(#4) ;
;@ letobjectw(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - current width ",N) ;
;@ letobjectw(N,"leftshoe.gif") ;
;@ notify("press Group 4 leftshoe.gif - current width ",N) ;
