substr()

Syntax:     substr (variable,source,start[,end])

The substr() statement isolates a substring from a source string and returns the resulting substring in the specified variable. The substring is isolated from the start character position to the end-1 character position. Character positions in a string start from 0. If the start character is less than zero or greater than or equal to the string length the result is an empty string. If the optional end variable is not specified or is less than the start character position then the substring returned is from the start position to the end of the source string. The source argument must be a string literal or a variable.

Related actions: concat(), replacestr(), strlen(), indexof()

 

Compatibility:

FKiSS 5   concat (S,"The"," quick brown ",'\n',"fox")  S becomes "The quick brown \nfox"

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@substr(A,"This is a string",0,4)  ; Variable A is set to"This"
;@substr(A,"This is a string",10) ; Variable A is set to "string"
;@substr(A,"This is a string",99) ; Variable A is set to ""
