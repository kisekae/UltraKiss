letcomment()

Syntax:     letcomment (variable,target)

The letcomment() statement returns the comment text associated with the cel or object in the specified variable.

The target can be a cel name, an object group specifier, or a cel group specifier. For object groups or cel groups the comment text associated with the group is the comment text for the first cel declared within the group. Comment text is specified in the KiSS configuration file when cel images are declared. Comment text always follows a semicolon on the cel declaration line.

Note that comment text does not contain any tag specifiers on the cel declaration, but does include all text following any tag specifiers.

 

 

Compatibility:

FKiSS 5   letcomment(S,"shirt.cel")  Variable S is set to the comment string for "shirt.cel"

 

UltraKiss Options:

AllAmbiguous apply test to all ambiguous cels default value "false" (Compatibility option)
ContextMap use page context to disambiguate cels default value "true" (Compatibility option)
VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letcomment(S,"shirt.cel") ; S contains the comment string
;@letcomment(N,#3) ; S contains the comment string of the first cel in #3
