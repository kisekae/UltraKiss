movie()

Syntax:     movie (filename[,repeat])

The movie() command will initiate the playback of a video cel. The filename may be a literal specification of a cel or a variable that contains a cel name.

Video cels are mapped when they are playing.

Multiple movie() commands can be concurrently active. A movie("") command with an empty filename will terminate all currently playing movie cels.

The optional integer repeat count indicates the number of times the movie should be played. A negative repeat count will result in countinuous repeated playback.

The FKiSS5 mediastart() and mediastop() events can also be used to control movie playback. These events fire on movie initiation and movie termination for the specified movie file.

Video cels also contain attributes that specify playback control information and presentation styles. Video cels attributes are:

controls  movie player controls are shown
repeat movie repeats indefinately
border border thickness, in pixels
bc background colour, integer RGB value
fc foreground colour, integer RGB value
bdc border colour, integer RGB value
fontname preferred font name, string
fontsize preferred font size, in points
fontstyle preferred font style:, normal, bold, italic

 

Related events: mediastart(), mediastop()
Related actions: mediaplayer(), getAttributes(), setAttributes()

 

Compatibility:

UltraKiss extension let(C,"mymovie.avi")
movie(C,-1)		  

 

Note: The UltraKiss movie() command is viewer specific and not compatible with other viewers.

 

UltraKiss Options:

CacheVideo cache video files in memory default value "false" (General option)
VariableCase variable names are case sensitive default value "false" (FKiSS option)

#2.999 button2.button : 0 ;%u %size[100,50] %attributes[text="Stop"]
#2.999 button1.button : 0 ;%size[100,50] %attributes[text="Play"]
#1 FOX.AVI            : 0 ;%size[320,240] %attributes[repeat]
