letcel() 

Syntax:     letcel (variable[,group[,frame]])

This action if specified as letcel(variable) stores the cel name of the last pressed cel in the variable. If the command is specified as letcel(variable,group) then the name of the first mapped cel in the group is returned in the variable. If the command is specified as letcel(variable,group,frame) then the cel name of the specified group frame is returned in the variable.

This command is typically used in a press event or other event to identify the currently selected cel.

The group parameter of the command must be a group object identifier. The frame is an integer value.

Related actions: letmapped()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ letcel(CelName)     ;
;@ notify("This cel last selected is ",CelName)    ;
