letmapped() 

Syntax:     letmapped (variable,object)

This action command identifies the number of visible or mapped cels in the object. The current page is the context for counting visible cels. Individual cels will return 1 if the specified cel is visible. A group or cel group object will return a number greater than 0 if any of the object cels are visible on the current page.

The object parameter of the command must be a cel, group object, or cel group identifier.

Related actions: letcel(), letcatch()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ letmapped(N,GroupName)     ;
;@ notify("The group object ",GroupName," has ",N," cels visible.")    ;
