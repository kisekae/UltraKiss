endwhile() 

Syntax:     endwhile (control)

This statement is the termination statement of a conditional loop. The control parameter must be a variable that matches to a previously executed 'while' statement. If there is no current 'while' statement for the control variable then the 'endwhile' statement is ignored and execution continues with the statement immediately following the 'endwhile' statement.. If a previous 'endwhile' statement was executed for the specified control variable then code execution is returned to the previous 'endwhile' statement for continuing execution.

If the previous 'while' loop conditional relationship is not true then code execution will continue with the statement immediately following the 'endwhile' statement.

Note that it is possible to have more than one 'endwhile' statement with the same control variable within one 'while' loop. However, at loop termination execution will continue with statements immediately following the last processed 'endwhile' statement.

 

Related actions: while(), for(), repeat()

;@ while(I)
;@  debug("I is ",I) 
;@  while(J)
;@   mul(X,I,J)
;@   debug("I is ",I," J is ",J)
;@   ifgreaterthan(X,8) let(J,0) endif()
;@   debug(" I*J is ",X)
;@  endwhile(J)
;@  ifgreaterthan(X,16) let(I,0) endif()
;@ endwhile(I)
