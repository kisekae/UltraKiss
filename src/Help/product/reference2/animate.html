animate() 

Syntax:     animate (target,period)

This is a new FKiSS5 action to animate the target entity. The target may be a literal name of an image file or a cel group name or an object group identifier. The animation time period is specified in milliseconds.

If the target is an image cel then the cel must be of a type that contains multiple frames, such as an animated GIF image. The animation period, if non-zero, starts the animation.

If the target is an object group or cel group, the animate() command will display each image frame in the group for the specified period. For cel groups with specified frames, the images are displayed in frame order. For cel groups without frames or object groups the images are displayed in cel specification order in the configuration file.

An animation period value of zero stops the target animation.


Related actions: map(), unmap(), setframe(), letframe()

 

Note: The UltraKiss animate() command is viewer specific and not compatible with other viewers.

 

 

Compatibility:

UltraKiss Extension animate("a.gif",1) or animate(CelGroup,250) or animate(#1,N) Animates all image frames in the group or animated image cel

 

UltraKiss Options:

AllAmbiguous apply animate to all ambiguous cels default value "false" (Compatibility option)
