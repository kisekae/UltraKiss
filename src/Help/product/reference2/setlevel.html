setlevel() 

Syntax:     setlevel (object,variable)

This action sets the drawing level of the specified KiSS object. This object can be a cel or group object. The drawing level of an object is initially set by the order in which the cel declaration appears in the CNF file. Cels that appear first are drawn on top of cels that appear lower in the order. The top order is number 0. The draw level of a group object is the minimum overlay level that needs to be considered when the group object is drawn. In other words it is the draw level with the largest number or the lowest most cel in the group.

The draw level can be used to move objects forwards and backwards in the draw hierarchy. This is another way to hide cels behind another background cel. Setting the draw level to 0 draws the cel on the top. Setting the draw level to a large number draws the cel on the bottom.

The object of the command must be a cel name specifier or a group object identifier. The variable is an integer value.

Related actions: letlevel(), letmapped()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ setlevel(N,"superman.cel")     ;
;@ notify("This cel is drawn at level ",N," in the draw hierarchy ")    ;
