repeat() 

Syntax:     repeat(labelname,count,iterator)

This statement repeats a label module for the specified number of times, incrementing the iterator variable each time. The 'repeat' statement was implemented with FKiSS 4.

The count parameter specifies the maximum repetition count and must be specified as an integer literal value or a variable. The repetition count is evaluated prior to first execution of the repeat() statement. Changes to the count variable value during execution will not affect the repetition count.

The iterator parameter must be a variable. The first repetition intializes this variable to the value 1 and each succeeding repetition increments the iterator by 1. On termination of the repeat() command the iterator variable will have the last iteration value used if the repeat() was terminated with an exitloop() command. Otherwise, the iterator variable will be set to the count value plus 1, unless DirectKiss compatibility mode is set. If in DirectKiss compatiblity the iterator will be set to the count value on loop termination.

Repeat statements can be nested. A repeat loop can be terminated early with the 'exitloop()' command.

The UltraKiss implementation of the repeat() statement is not reentrant.

 

Related actions: exitloop(), for(), repeat(), while()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ let(I,5)
;@ repeat(label1,I,N)

;@ label(label1)
;@ ifequal(N,1) debug("Iteration 1") endif()
;@ ifequal(N,2) debug("Iteration 2 ") endif()
;@ ifequal(N,3) debug("Iteration 3 exitloop()") exitloop() endif()
;@ ifequal(N,4) debug("Iteration 4 ") endif()
;@ ifequal(N,5) debug("Iteration 5 ") endif()
