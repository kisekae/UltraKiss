exitloop() 

Syntax:     exitloop ()

The exitloop()statement will terminate any repetition of a label module invoked through a repeat() statement. If the label module has not been invoked through a repeat() statement then the exitloop() command has no effect.

 

Related actions: repeat(), while(), for()

 

 Example: 

;@ let(I,5)
;@ repeat(label1,I,N)

;@ label(label1)
;@ ifequal(N,1) debug("Iteration 1") endif()
;@ ifequal(N,2) debug("Iteration 2 ") endif()
;@ ifequal(N,3) debug("Iteration 3 exitloop()") exitloop() endif()
;@ ifequal(N,4) debug("Iteration 4 ") endif()
;@ ifequal(N,5) debug("Iteration 5 ") endif()
