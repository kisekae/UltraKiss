never() 

Syntax:     never ()

This event is never triggered. It can be used as a placeholder to hide events or event code that may not be recognized by other viewers.

Other names for the never() event are unknown() and nothing(). 

The intention of never() was to wrap unknown event statements that might not be understood by earlier viewers, such as version(), so that the earlier viewers would not complain when parsing the FKiSS code.

 

Example:

;@never()               ; hides the following event from viewers that do not understand version()
;@version(3)            ; fired if the viewer supports FKiSS3
;@ unmap("fk3warn.cel") ; a warning that FK3 is needed

 

One difficulty with never() is that it does not work for unknown FKiSS action statements. KiSS viewers are supposed to ignore actions that they do not understand, but many viewers will issue warning messages when unknown actions are parsed.
