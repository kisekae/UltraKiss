setSelectedValue() 

Syntax:     setSelectedValue(component,value)

This action selects the item in the list component or combobox component providing that the string value is a member of the list. The component may be a literal name of a list or or a variable that contains the name of a list component. The value may be a string literal specification or a variable that has a string value.

The comparision for selection is case sensitve. The item must be in the list exactly as specified for it to be selected. For list components that have the 'multiple' attribute set to allow multiple selections, the setSelectedValue() command will allow for multiple selections if applied to different items in the component.

This command fails silently if the specified component is not a list component or a combobox component.

Components are declared as cels within a KiSS configuration as shown in the example below.  Component cel identifiers use specific extensions such as .button, .list, or .textarea to identify the component type.  

#1.9999 List-1.List           : 0     ;%size[146,207] %offset[12,121] %attributes[wrap,vsb]
#1.9999 TextArea-1.TextArea   : 0     ;%size[269,49] %offset[227,176] %attributes[wrap]
#1.9999 Button1.Button        : 0     ;%size[135,25] %offset[325,267] %attributes[text="Press Me"]

The example above defines a List component with vertical scroll bars, a TextArea component that performs word-wrap, and a Button.  The size and location of these components within the object are specified with the %size[] and %offset[] tags.  Component attributes are specified with the %attributes[] tag.  

 

Related actions: getSelectedValue(), getSelectedIndex(), getNextSelectedIndex(), setSelectedIndex(), getSelected(), setSelected(), getSelectedItem(), setSelectedItem()

 

Compatibility:

UltraKiss Extension setSelectedValue ("list1.list","new value") sets the first selected item in the list to the "new value"

 

Note: The UltraKiss setSelectedValue() command is viewer specific and not compatible with other viewers.

 

Example: 

#8.999 button1.button         : 0 ;%size[318,26]     %attributes[text="Press Me!"] 
#3.999 list1.list             : 0 ;%size[175,177]    %attributes[vsb]
