next() 

Syntax:     next (control)

This statement is the termination statement of an iteration loop.

The control parameter must be a variable that matches to a previously executed 'for' statement. If there is no current 'for' statement for the control variable then the 'next' statement is ignored and execution continues with the statement immediately following the 'next' statement.. If a previous 'for' statement was executed for the specified control variable then code execution is returned to the previous 'for' statement for continuing execution.

If the previous 'for' loop is satisfied such that loop termination occurs then code execution will continue with the statement immediately following the 'next' statement.

Note that it is possible to have more than one 'next' statement with the same control variable within one 'for' loop. However, at loop termination execution will continue with statements immediately following the last processed 'next' statement.

A 'for' loop can be terminated early by assigning the loop end value to the control variable.

 

Related actions: for(), while(), repeat()
