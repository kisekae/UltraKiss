getSelectedItem() 

Syntax:     getSelectedItem(variable,component)

This is a component command to retrieve the value of a combobox component that has a selected item. The component may be a literal name of a combobox, or a variable that contains the name of a combobox component. The value of a selected item is the string representation of the item in the list. Note that the action command name 'getSelectedItem' is case sensitive and must be specified as shown.

This command sets the specified variable to the string value of the first selected item index in the combobox. If there is no currently selected item in the combobox or if the component is not a combobox the variable is set to an empty string. Note that indexing of list items begins at 0.

Components are declared as cels within a KiSS configuration as shown in the example below.  Component cel identifiers use specific extensions such as .button, .list, or .textarea to identify the component type.  

#1.9999 List-1.List           : 0     ;%size[146,207] %offset[12,121] %attributes[wrap,vsb]
#1.9999 TextArea-1.TextArea   : 0     ;%size[269,49] %offset[227,176] %attributes[wrap]
#1.9999 Button1.Button        : 0     ;%size[135,25] %offset[325,267] %attributes[text="Press Me"]

The example above defines a List component with vertical scroll bars, a TextArea component that performs word-wrap, and a Button.  The size and location of these components within the object are specified with the %size[] and %offset[] tags.  Component attributes are specified with the %attributes[] tag.  

 

Related actions: setSelectedItem(), setSelectedValue(), getSelectedValue(), setSelectedIndex(), getSelectedIndex(), getNextSelectedIndex(), setSelected(), getSelected()

 

Compatibility:

UltraKiss Extension getSelectedItem(S,"combo1.combobox") Sets the variable S to the string value of the first selected item in the list

 

Note: The UltraKiss getSelectedItem() command is viewer specific and not compatible with other viewers.

 

Example: 

#8.999 button1.button         : 0 ;%size[318,26]     %attributes[text="Press Me!"] 
#3.999 list1.list             : 0 ;%size[175,177]    %attributes[vsb] 

;@initialize() ;
;@ addItem("list1.list","carrots") ; index 0

;@ addItem("list1.list","peas") ; index 3
;@ addItem("list1.list","apple, orange, grape, cherry, plum, pear, peach") ;
