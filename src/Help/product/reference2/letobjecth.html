letobjecth()

Syntax:     letobjecth (variable,target)

The letobjecth() statement returns the current height of a cel or object on the currently visible page.

The same object on different page can have a different height.

The target can be a cel name or an object group specifier. FKiSS 3 allowed only object groups for the letobjecth() statement, FKiSS 5 allows cel names.

 

Related actions: letobjectw() letobjectx() letobjecty()

 

 

Compatibility:

FKiSS 5   letobjecth(N,"shirt.cel")  Variable N is set to the current height of "shirt.cel".

 

Note: The UltraKiss letobjecth() command is viewer specific and not compatible with other viewers.

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letobjecth(N,"shirt.cel") ; N contains the current height for the specified cel
;@letobjecth(N,#3) ; N contains the current height for object #3

 

 Example: 

;[Cel Section]
#4 rightshoe.gif : 0 ;
#4 leftshoe.gif : 0 ;
#4 sailormoon.gif : 0 ;
 
;[FKiSS Section]
;@EventHandler
 
;@press(#4) ;
;@ letobjecth(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - current height ",N) ;
;@ letobjecth(N,"leftshoe.gif") ;
;@ notify("press Group 4 leftshoe.gif - current height ",N) ;
