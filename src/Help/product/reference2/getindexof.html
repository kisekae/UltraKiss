getIndexOf() 

Syntax:     getIndexOf(variable,component,value)

This is a component command to retrieve the index of a list component that has the specified value. The component may be a literal name of a list component or a combobox component or a variable that contains the name of a list component or combobox component. The value is a literal string or a variable that contains the string representation of the desired item. Note that the action command name 'getIndexOf' is case sensitive and must be specified as shown.

This command returns the integer value of the item in the list that has the specified value, and -1 if the item cannot be found or the component is not a list or combobox. Value comparisons are case sensitive. Note that list index values begin at 0.

Components are declared as cels within a KiSS configuration as shown in the example below.  Component cel identifiers use specific extensions such as .button, .list, or .textarea to identify the component type.  

#1.9999 List-1.List           : 0     ;%size[146,207] %offset[12,121] %attributes[wrap,vsb]
#1.9999 TextArea-1.TextArea   : 0     ;%size[269,49] %offset[227,176] %attributes[wrap]
#1.9999 Button1.Button        : 0     ;%size[135,25] %offset[325,267] %attributes[text="Press Me"]

The example above defines a List component with vertical scroll bars, a TextArea component that performs word-wrap, and a Button.  The size and location of these components within the object are specified with the %size[] and %offset[] tags.  Component attributes are specified with the %attributes[] tag.  

 

Related actions: setValueAt(), getValueAt(), getItemCount()

 

Compatibility:

UltraKiss Extension getIndexOf (N,"list1.list","corn") Sets the variable N to the index

 

Note: The UltraKiss getIndexOf() command is viewer specific and not compatible with other viewers.

 

Example: 

#8.999 button1.button         : 0 ;%size[318,26]     %attributes[text="Press Me!"] 
#3.999 list1.list             : 0 ;%size[175,177]    %attributes[vsb] 

;@initialize() ;
;@ addItem("list1.list","carrots") ; index 0
;@ addItem("list1.list","potatoes") ; index 1
;@ addItem("list1.list","corn") ; index 2
;@ addItem("list1.list","peas") ; index 3
;@ addItem("list1.list","apple, orange, grape, cherry, plum, pear, peach") ;

;@press(#8) ;
;@ getIndexOf(N,"list1.list","corn") ; N set to 2
;@ getIndexOf(N,"list1.list","Corn") ; N set to -1
