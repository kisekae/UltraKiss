letkcf() 

Syntax:     letkcf (variable,target)

This action returns the ordinal palette number (kcf file number) that is being used for a cel, group, or celgroup. Note that palette file numbers begin from 0.

If the cel does not reference a palette file the value returned is -1.

If the group object or cel group member image cels do not all reference the same palette file the value returned is -1.

The target can be a cel name or an object group specifier or a cel group specifier.

Related actions: letpal(), setkcf(), setpal()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ letkcf(N,"256-col1.cel")     ;
;@ notify("This cel references palette file ",N)    ;
