getSelected() 

Syntax:     getSelected (variable,component)

This is a component command to retrieve the selected state of an abstract button component. The component may be a literal name of a button or togglebutton or menu item. Note that the action command name 'getSelected' is case sensitive and must be specified as shown.

Buttons, when pressed, are considered to be selected and when released are unselected. Toggle buttons maintain state and each time a toggle button is pressed the component will switch between a selected state and an unselected state. Menu items are also treated like a button and when pressed are in a selected state.

This command returns an integer valueof 1 if the component is in a selected state, 0 if not in a selected state, and -1 if the specified component does not maintain selection states.

Components are declared as cels within a KiSS configuration as shown in the example below.  Component cel identifiers use specific extensions such as .button, .list, or .textarea to identify the component type.  

#1.9999 List-1.List           : 0     ;%size[146,207] %offset[12,121] %attributes[wrap,vsb]
#1.9999 TextArea-1.TextArea   : 0     ;%size[269,49] %offset[227,176] %attributes[wrap]
#1.9999 Button1.Button        : 0     ;%size[135,25] %offset[325,267] %attributes[text="Press Me"]

The example above defines a List component with vertical scroll bars, a TextArea component that performs word-wrap, and a Button.  The size and location of these components within the object are specified with the %size[] and %offset[] tags.  Component attributes are specified with the %attributes[] tag.  


Related actions: setSelected(), getSelectedIndex(), getNextSelectedIndex(), setSelectedIndex(), getSelectedValue(), setSelectedValue(), getSelectedItem(), setSelectedItem()

 

Compatibility:

UltraKiss Extension getSelected (On,"button1.togglebutton") Sets the 'On' variable to 1 if the toggle button has been selected

 

Note: The UltraKiss getSelected() command is viewer specific and not compatible with other viewers.

 

Example: 

#3.999 button1.button         : 0 ;%size[318,26]     %attributes[text="Press Me!"] 
#3.999 list1.list             : 0 ;%size[175,177]    %attributes[vsb] 

;@press("button1.button") ; 
;@ getSelected(N,"button1.button") ; 
;@ notify("button selection is ",N) ;
