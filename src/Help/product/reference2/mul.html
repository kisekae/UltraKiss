mul()

Syntax:     mul (result,operand1,operand2)

This command performs an arithmetic multiplication. The result variable is is assigned the product of operand1 and operand2.  The result must be a variable. Operand1 and operand2 can be numeric literal values or variables that represent numeric values.

result = operand1 * operand2. 

FKiSS 3 and FKiSS 4 only support integer values that range from -2147483648 to 2147483647, inclusive.  UltraKiss FKiSS 5 provides support for floating point numbers and long integers that can range from -9223372036854775808 to 9223372036854775807, inclusive. With FKiSS5 implicit conversion between variables is automatically performed and all calculations maintain the appropriate precision. 

Numeric overflow may occur if the resulting difference cannot be accurately represented by the computer. 

Related actions: add(), sub(), div(), mod(), math(), sqrt(), random()

 

Compatibility:

FKiSS 3 mul(X,A,B)  or  mul(X,A,3) integer multiplication, single letter variable names
FKiSS 5 mul(MyVariable,A,3.145) or mul(X,A,999999999999) integer, string, floating point multiplication, long variable names

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

Example Code (FKiSS3): 

;@let(A,1) let(B,2) let(C,3)                         ; 
;@mul(X,A,B)                                         ; simple multiplication
;@mul(X,C,-1)                                        ; literal constants

Example Code (FKiSS5): 

;@let(Apple,1) let(Pear,2) let(PI,3.1415927)         ; 
;@mul(TwoPie,PI,PI)                                  ; floating point multiplication
;@mul(OnePie,TwoPie,-3.1415927)                      ; floating point constant
;@mul(BigPie,OnePie,-1234567890123456)               ; long constant
;@mul(X,4,2)                                         ; integer arithmetic
