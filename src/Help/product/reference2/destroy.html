destroy()

Syntax:     destroy (object)

The destroy() statement destroy the specified group object. The object group is removed from all pages on which it appears. All cels within the group are also destroyed. The group object is removed from the configuration.

It follows that if the object is removed from the configuration and the configuration is saved or reloaded that new object numbers could be assigned when the configuration is processed. This could evaluate existing FKiSS code. Accodingly, destroyed objects should only be objects that have been previously cloned.

The object must be an object group specifier or a variable containing an object group identifier.

 

Related actions: clone()

 

 

Compatibility:

FKiSS 5   destroy(#3)  Group object #3 is destroyed.

 

Note: The UltraKiss destroy() command is viewer specific and not compatible with other viewers.

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@destroy(N,"shirt.cel") ; N contains the current height for the specified cel
;@destroy(N,#3) ; N contains the current height for object #3
