mouseRelease() 

Syntax:     mouseRelease()

The mouseRelease() command forces a release of the mouse from user control. On a mouse release all release(), drop(), collision, and unfix() events are processed.

 

Compatibility:

UltraKiss Extension mouseRelease() forces a mouse release

 

Note: The UltraKiss mouseRelease() command is viewer specific and not compatible with other viewers.

 

 Example: 

;@drop(#3) ;
;@ ifequal(Released,1) map(#4) endif() ;
;@ timer(Reset,1000) ;

;@press(#3) ;
;@ timer(Pressed,2000) ;
;@ let(Released,0) ;
;@ unmap(#4) ;

;@alarm(Pressed) ;
;@ let(Released,1) ;
;@ mouseRelease() ;

;@alarm(Reset) ;
;@ unmap(#4) ;
