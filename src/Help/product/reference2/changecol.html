changecol() 

Syntax:     changecol (integer)

This action sets the color set or palette group to be used when displaying all cels on the current page. KiSS palette files can contain many different color sets and each cel image is associated with a specific palette or KCF file. Every page within the set configuration is associated with a default palette group number. When multiple palettes are used within KCF files, this feature allows for different color variations to be applied to various images on different pages.

Palette groups, or multipalettes, begin from 0. Note that the changecol() command will fire all col() events defined for the specified color set.

Not all KCF files are required to have multiple palette groups defined. If an image cel cel is associated with a palette file that does not contain the identified palette group then this image cannot change colors. In this case the image will be displayed using the colors found in the last palette group of the associated palette file.

Truecolor cels, or Cherry Kiss cels, do not have an associated palette file and are not affected by a changecol() command.

The changecol() command should not be used in initialize() events, but can be used in begin() events.

 

UltraKiss Extension: 

UltraKiss introduced the capability to fix a specific multipalette specification to a specific image cel. This fixed specification is provided on the cel definition line in the set configuration (CNF) file. The optional <Palette group> specifier is an integer number that fixes the multipalette to be used when displaying the image. When fixed palette groups are specified the image will not change colors through a changecol() command.

Syntax:     #<Object>[.<Fix>] <Cel filename> [*<Palette number>[.<Palette group >]] [:<Page Set number>...]
                        [; [<%Tag>...] [<!Cel group [: <frame ...>] ...] comment]


Related actions: col(), changeset(), letmaxcolor()

 

Compatibility:

FKiSS 1 allows cel literal names or objects. changecol (3) Sets the color mapping to the fourth multipalette for all images using KCF files.
FKiSS 3 allows variables for objects.   let(X,10) changecol (X)  Sets the color mapping of the all cels to the multipalette specified by variable X.
FKiSS 5 allows an unlimited number changecol(25) Previous FKiSS versions were limited to a maximum 10 palette groups.

 

 Example: 

%palette0.kcf

#1.999 button.cel : 0 ;      
#3.999 fk4b3a.cel : 0 ;     

;@changecol(3)    ;
