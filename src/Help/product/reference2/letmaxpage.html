letmaxpage() 

Syntax:     letmaxpage (variable)

This action returns the maximum number of pages currently defined in the KiSS set. Note that page numbers begin from 0. Note that with FKiSS 5 an unlimited number of pages can be created for a KiSS set.

Related actions: set(), changeset(), letmaxcolor()

 

UltraKiss Options:

MaxPageSet specifies the maximum number of pages displayed on the Ultrakiss toolbar. This is not the maximum number of pages that can be defined for the set. default value "10" (Compatibility option)

 

 Example: 

;@letmaxpage(N)    ;
;@notify("This set has ",N," pages")    ;
