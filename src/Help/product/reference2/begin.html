begin()

Syntax: begin()

This event is triggered by opening the set, after initialize() and version(). This event fires after the cels on the first page have been mapped.

The begin() event is often used to initiate alarms and activate sounds when the set first becomes visible. The begin() event is an immediate event and will complete execution before any other event starts. More than one begin() event can be specified. If sequential event processing is enforced each event should fire in the sequence in which it was defined. Cels that are mapped and moved within the begin() event can fire collision events. 

The begin() event will be processed before any set() events or col() events are fired for the starting page. Note that there is no defined order for event processing in KiSS, so any logic coded that assumes one event has completed before another starts becomes viewer dependent. However, it is usually safe to assume that initialize(), version(), and begin() events are processed sequentially and completely during set initialization.


Related events: initialize(), end(), version()

 

UltraKiss Options:

ImmediateEvent  enforces sequential event processing   default value "false" (Compatibility option)
TimerPeriod the polling period in milliseconds for firing alarms default value "10" (FKiSS option)
EventQueues  the number of concurrent events that can run  default value "1" (FKiSS option)
TimerOption enables timer processing default value "true" (FKiSS option)
EventOption  enables alarm processing   default value "true" (FKiSS option)

 

 

FKiSS 1 Example:

#9 button1.cel  :0         ; An overlay picture

;@begin()                  ; 
;@ sound("smc181-2.wav")   ; 
;@ timer(1,500)            ; 
;@ timer(2,4000)           ; 

;@alarm(1)                 ; Fade the overlay
;@ transparent(#9,3)       ; 
;@ timer(1,40)             ; 

;@alarm(2)                 ; Erase the overlay
;@ unmap(#9)               ; 
;@ timer(1,0)              ; 

 

FKiSS 2 Example:

#12 label4.cel :0          ; apart messages
#12 label3.cel :0          ;
#4 rboot.cel   :0          ;
#4 lboot.cel   :0          ;
#1 smoon.cel   :0          ;

;@initialize()             ; 
;@ unmap("label4.cel")     ; 

;@begin()                  ; 
;@ sound("smc181-2.wav")   ; Begin sound
;@ timer(1,1)              ; 
;@ timer(2,3500)           ; 
;@ movebyx(#4,#1,140)      ; 

;@alarm(1)                 ; Fade the overlay
;@ transparent(#9,3)       ; 
;@ timer(1,40)             ; 

;@alarm(2)                 ; Stop the fade
;@ unmap(#9)               ; 
;@ timer(1,0)              ; 

;@apart("lboot.cel","smoon.cel") ; movebyx apart
;@ unmap("label3.cel")     ; 
;@ map("label4.cel")       ;
