letkeychar()

Syntax:     letkeychar (variable)

The letkeychar() statement sets the variable to the character string of the last key pressed.

The letkeychar() startement will recognize the 'enter' key as a newline character. All other control characters are returned as an empty character string. To identify control character keyboard input use the getkeycode() command.

The letkeychar() statement does not distinguish between upper and lower case characters unless the UltraKiss keyboard case option is set. The letkeychar() statement will respond to all keys on the keyboard.

Some viewers may only recognize keyboard entry while a key is actually being pressed. This can lead to viewer compatibility problems as the letkeychar() statement may only function within keypress() events.

 

Related events: keypress(), keyrelease(), keytype()
Related actions: letkeymap(), letkeychar(), letkeycode(), letkeystring(), letkeymodifier()

 

UltraKiss Options:

KeyCase Enable keyboard upper/lower case key entry default value "false" (FKiSS option)
RetainKey Remember last key pressed default value "true" (Compatibility option)

 

 Example: 

;@letkeychar(C) ; Sets variable C to the character typed, or an empty string
