setkcf() 

Syntax:     setkcf (target,variable)

This action sets the ordinal palette number (kcf file number) to be used for a cel, group, or celgroup. Note that palette file numbers begin from 0.

If the new palette file contains multipalettes then the corresponding multipalette colors will be applied to the cel. If the new palette file does not have enough multipalettes or is not compatible then no color change will occur to the cel image.

For group objects or cel groups the palette file change is applied to all image cels within the object.

The target can be a cel name or an object group specifier or a cel group specifier. The variable must evaluate to be an integer value.

Related actions: letpal(), letkcf(), setpal()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ setkcf("256-col1.cel",N)     ;
;@ notify("This cel now references palette file ",N)    ;
