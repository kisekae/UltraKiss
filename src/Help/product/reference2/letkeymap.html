letkeymap()

Syntax:     letkeymap (variable,keyset)

The letkeymap() statement sets the variable to the binary conjunction of character indexes in the keyset string of the last set of keys simultaneously pressed, or the value 0 if no key is not in the list. The letkeymap() statement is designed to detect simultaneous key combinations.

The keyset parameter can be a string literal or a variable that contains a string. Each character in the string should represent one possible character on the keyboard that can be tested. For example, the string "abcxyz" is a set of six possible characters that are to be tested for keyboard input.

The value returned in the variable is a bit string where the bit in position 'n', if set, represents that the character in the keyset at index 'n' was pressed. For example, if the keyset was "abcdef" and keys 'b' and 'd' were simultaneously pressed, then the bits 2 and 4 would be set resulting in the number 10 being returned in the variable. 10 = 2 + 8.

The letkeymap() startement will also recognize the four cursor control keys 'up', 'down', 'left', and 'right' as special entries within the keyset. For example, the keyset "abcrightleft" recognizes the first three letters of the alphabet plus two cursor control keys, with index values 1, 2, 3, 4, and 9.

The letkeymap() statement does not distinguish between upper and lower case characters unless the UltraKiss keyboard case option is set. The letkeymap() statement is also not capable of recognizing most special keys on the keyboard.

Some viewers may only recognize keyboard entry while a key is actually being pressed. This can lead to viewer compatibility problems. If the letkeymap() statement is placed within keypress() events then many event executions may occur as each key in the multikey combination is pressed. This can lead to progressively increasing values being returned at the time that keys are concurrently pressed.

This oddity with cursor keys illustrates the original design problems with FKiSS.

 

Related events: keypress(), keyrelease(), keytype()
Related actions: letkey(), letkeychar(), letkeycode(), letkeystring(), letkeymodifier()

 

UltraKiss Options:

KeyCase Enable keyboard upper/lower case key entry default value "false" (FKiSS option)
RetainKey Remember last key set pressed default value "true" (Compatibility option)

 

 Example: 

;@letkeymap(N,"a") ; Sets variable N to 1 if the "a" key is pressed, 0 otherwise
;@letkeymap(N,"012345xyzabcupdown") ; Sets variable N to 1025 if the "a0" key is pressed
