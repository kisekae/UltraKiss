while() 

Syntax:     while(operand1[,operand2[,condition]])

This statement specifies the beginning of a conditional loop. The 'while' statement operand1 parameter value is compared to the operand2 parameter value according to the specified condition. While the relation is true the code statements following the 'while' statements up to the first following 'endwhile' statement are repeatedly processed. The code statements that follow are not executed if the relation is false.

Operand1 must be specified as a variable. This variable is a control variable used to match the 'while' statement to corresponding 'endwhile' statements.

If the optional operand2 value is not specified then the operand1 value can be considered as a boolean switch, where a non-zero or "true" value for operand1 will cause the code following to be repeatedly processed, and a zero or any other value will terminate the loop and not execute the code.

Operand2, if specified for comparisons, can be a literal value or a variable.

If the optional operand3 value is not specified then the condition for comparison defaults to equality. The code following will be repeatedly processed while the operand1 value equals the operand2 value. If operand3 is specified then it should be a string literal value or a variable with a value of "=", "<", ">", "<=", ">=", "<>", "!=", which compare for equality, strictly greater than, strictly less than, greater than or equal to, less than or equal to, or not equal.

Operand comprisons are made according to the data type of each operand. String comparisons are performed if either operand is of string type. Floating point numeric comparisons are performed if either operand is a floating point number. Long number comparisons are performed if either operand is of a long number data type.

It is possible to nest 'while' loops providing that each 'while' loop uses a unique control variable for operand1.

 

Related actions: endwhile(), for(), repeat()
