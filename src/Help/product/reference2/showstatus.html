showstatus() 

Syntax:     showstatus (string[,string,string,...])

The showstatus() action displays a message on the viewer status bar. The arguments specify the text to be displayed in the status line and can be variable or literal strings. Each argument is converted to its string representation and concatenated with all other arguments in the order specified. 

To set the status line to its default state, use showstatus() without any arguments.

Related actions: debug(), notify(), confirm()

 

Compatibility:

UltaKiss extension showstatus("Variable X is: ",X) Displays a composite message in the system status bar.

 

Note: The UltraKiss showstatus() command is viewer specific and not compatible with other viewers.

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 FKiSS 5 Example:

#1.999 button.cel : 0 ;      
;@press(#1)   let(S,"Object #1 pressed") showstatus(S) ; 
;@release(#1) showstatus(S," and then released")  ;
