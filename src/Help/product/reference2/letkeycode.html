letkeycode()

Syntax:     letkeycode (variable)

The letkeycode() statement sets the variable to the virtual character code of the last key pressed.

The letkeycode() startement always returns a numeric value for any key pressed. The letkeycode() statement does not distinguish between upper and lower case characters unless the UltraKiss keyboard case option is set. The letkeycode() statement will respond to all keys on the keyboard.

Some viewers may only recognize keyboard entry while a key is actually being pressed. This can lead to viewer compatibility problems as the letkeycode() statement may only function within keypress() events.

Related events: keypress(), keyrelease(), keytype()
Related actions: letkeymap(), letkeychar(), letkeychar(), letkeystring(), letkeymodifier()

 

UltraKiss Options:

KeyCase Enable keyboard upper/lower case key entry default value "false" (FKiSS option)
RetainKey Remember last key pressed default value "true" (Compatibility option)

 

 Example: 

;@letkeycode(N) ; Sets variable N to the virtual character code of the character typed
