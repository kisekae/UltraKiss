letkeymodifier()

Syntax:     letkeymodifier(variable)

The letkeymodifier() statement sets the variable to a character string identifying the modifier keys pressed on the last key entry. Modifier keys are the 'shift' key, 'alt' key, and 'ctrl' key.

The returned values are strings concatenated with a plus (+) sign when multiple modifier keys are concurrently pressed. If no modifier characters are used then an empty string is returned.

The letkeymodifier() statement is typically used in conjunction with the letkeychar() statement to isolate modifer values on key input.

Some viewers may only recognize keyboard entry while a key is actually being pressed. This can lead to viewer compatibility problems as the letkeychar() statement may only function within keypress() events.

 

Related events: keypress(), keyrelease(), keytype()
Related actions: letkeymap(), letkeychar(), letkeycode(), letkeystring(), letkeymodifier()

 

UltraKiss Options:

KeyCase Enable keyboard upper/lower case key entry default value "false" (FKiSS option)
RetainKey Remember last key pressed default value "true" (Compatibility option)

 

 Example: 

;@letkeymodifier(M) ; Sets variable M to the modifier string, or an empty string
