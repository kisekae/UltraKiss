format() 

Syntax:     format (variable,number,maxfraction,minfraction,maxint,minint)

This action command formats a number into a string variable with the specified number of digits to the right and the left of the decimal point.

The number parameter can specify any type of numeric value. This command is typically used to format real numbers for output.

The maxfraction and minfraction parameters can be integer literal values or variables that specify the maximum and minimum number of digits that can appear in the fractional portion of the number. The maxfraction value must be greater than or equal to the minfraction value.

The maxint and minint parameters specify the maximum and minimum number of digits that can appear in the integer portion of the number. The maxint value must be greater than or equal to the minint value. These parameter values can be integer literal values or variables that have integer values.

 

Related actions: strlen() , indexof(), substr(), replacestr(), concat()

 

Compatibility:

FKiSS 5   format (S,3.14159,2,2,8,0)  Formats the value of PI to 3.14 as a string in S

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)
