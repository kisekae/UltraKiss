letset() 

Syntax:     letset (variable)

This action obtains the current set number, also known as a page that is being displayed. The variable is an integer value.

Related actions: changeset(), changecol(), setkcf()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ letset(N)     ;
;@ notify("This page set is currently on display ",N,")    ;
