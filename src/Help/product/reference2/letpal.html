letpal() 

Syntax:     letpal (variable,cel)

This action obtains the multipalette number, also known as a palette group in a cel palette file (KCF file) that is being used for coloring a cel. Note that multipalette file numbers begin from 0.

KiSS palette files can contain up to 10 different color palettes that are either 16 color or 256 colors in size. The optional palette group specification is an UltraKiss extension in the CNF file to identify the fixed palette group of colors that will be applied to the image. This fixed palette group will not change as color sets change and is related to the letpal() and letpal() FKiSS action commands.

The object of the command must be a cel name specifier. The variable is an integer value.

Related actions: setpal(), letkcf(), setkcf()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ letpal(N,"256-col1.cel")     ;
;@ notify("This cel is referencing multipalette ",N," in the cel palette file ")    ;
