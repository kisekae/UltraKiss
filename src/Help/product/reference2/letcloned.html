letcloned()

Syntax:     letcloned (variable,object)

The letcloned() statement returns the object number of the original source object from which the specified object was cloned. The object is an integer. A value of -1 is returned if the specified object is not a clone of another object.

The object must be an object group specifier or a variable containing an object group identifier.

 

Related actions: clone() destroy()

 

 

Compatibility:

FKiSS 5   letcloned(N,#3)  Variable N is set to the group object number that was the source object for the cloning of object #3.

 

Note: The UltraKiss letcloned() command is viewer specific and not compatible with other viewers.

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;[Cel Section]
#2 button1.button : 0 ;%size[151,50] %attributes[text="Clone Sailormoon"]
#1 sailormoon.cel : 0 ;

;[Page Set Section]
; Page 0
$0 * 284,278 81,78 * * *

;[FKiSS Section]
;@EventHandler

;@press("button1.button") ;
;@ clone(NewGroup,#1) ;@ letcloned(Source,NewGroup) ;
;@ notify("Cloned group is ",NewGroup," source object is ",Source) ;
