letheight()

Syntax:     letheight (variable,target)

The letheight() statement returns the visible height of an object in pixels in the specified variable.

The visible height is context sensitive to the page that is currently visible. The same object on different page can have a different height.

The target can be an object group specifier.

 

Related actions: letwidth()

 

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letheight(N,#3) ; N contains the visible height for object #3

 

 Example: 

;[Cel Section]
#4 rightshoe.gif : 0 ;
#4 leftshoe.gif : 0 ;
#4 sailormoon.gif : 0 ;
 
;[FKiSS Section]
;@EventHandler
 
;@press(#4) ;
;@ letheight(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - height ",N) ; Answer is 192
;@ unmap("sailormoon.gif") ;
;@ letheight(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - height ",N) ; Answer is 28
