letparent()

Syntax:     letparent (variable,object)

The letparent() statement returns the parent object in the attachment chain for the specified object.

If no parent object exists the variable is set to -1.

The object must be an object group specifier or a variable containing an object group identifier.

 

Related actions: attach() detach() letsiblingx() letchild()

 

 

Compatibility:

FKiSS 5   letparent(N,#3)  Variable N is set to the only parent of group object 3.

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letparent(N,"shirt.cel") ; N contains the current height for the specified cel
;@letparent(N,#3) ; N contains the current height for object #3

 

 Example: 

;[Cel Section]
#4 rightshoe.gif : 0 ;
#4 leftshoe.gif : 0 ;
#4 sailormoon.gif : 0 ;
 
;[FKiSS Section]
;@EventHandler
 
;@press(#4) ;
;@ letparent(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - current height ",N) ;
;@ letparent(N,"leftshoe.gif") ;
;@ notify("press Group 4 leftshoe.gif - current height ",N) ;
