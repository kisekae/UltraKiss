altmap() 

Syntax:     altmap (target)

This action inverts the mapping or visibility of the target. The target may be a literal name of an image file or a cel group name or an object group identifier.

For object groups or cel groups, each image cel in the group is inverted separately.

FKiSS adjustment of cel visibility through altmap() commands can trigger collision events if the viewer supports visibility collisions. For example, an image collide() event could fire when the cel becomes visible, or an apart() event could fire when the cel was hidden.

Altmap() is a binary toggle generally useful for objects with one or two image cels. If the object contains only one cel, altmap() is a visibility switch for the object. If the object contains two image cels where one cel is unmapped and the other cel is mapped, altmap() becomes a toggle for alternately showing each image. For objects that contain more than two cels, altmap() capability is better managed by cel group frames for individual image frame mapping. 


Related actions: map(), unmap(), setframe(), letframe(), ghost()

 

Compatibility:

FKiSS 1 allows cel literal names or objects. altmap ("cel1.cel") or altmap(#1) Inverts the mapping of the specified cel, or all cels in the object group.
FKiSS 3 allows variables for objects.   let(X,10) altmap (X)  Inverts the mapping of the all cels in the object group specified by variable X.
FKiSS 4 allows cel group names.   altmap (!Table)  Inverts the mapping of all cels in the cel group.
FKiSS 5 allows variables for all names let(Cel,"a.cel") altmap (Cel)  Inverts the mapping of the specified cel, or all cels in the object group or cel group.

 

UltraKiss Options:

AllAmbiguous apply altmap to all ambiguous cels default value "false" (Compatibility option)
MapCollide apply collision events on visibility changes default value "false" (Compatibility option)

 

MouseOver Code Example (FKiSS4): 

#3.999 fk4b3a.cel : 0 ;      normal image
#3.999 fk4b3b.cel : 0 ;%u    highlight image
;@mousein(#3) altmap(#3)  ; 
;@mouseout(#3) altmap(#3) ; 


ToggleButton Code Example (FKiSS1): 

#13.999 fk4b5a.cel : 0 ;    button released image   
#13.999 fk4b5b.cel : 0 ;    button pressed image

;@initialize()         ;
;@ unmap("fk4b5b.cel") ;

;@press("fk4b5a.cel")  ; press toggle
;@ altmap(#13)         ; 

;@press("fk4b5b.cel")  ; release toggle
;@ altmap(#13)         ;
