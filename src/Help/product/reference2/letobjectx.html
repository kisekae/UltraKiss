letobjectx()

Syntax:     letobjectx (variable,target)

The letobjectx() statement returns the current X ordinate of a cel or object on the currently visible page.

The same object on different page can have a different position.

The target can be a cel name or an object group specifier. FKiSS 3 allowed only object groups for the letobjectx() statement, FKiSS 5 allows cel names.

 

Related actions: letobjecty() letinitx() letinity()

 

 

Compatibility:

FKiSS 5   letobjectx(N,"shirt.cel")  Variable N is set to the current X ordinate of "shirt.cel". This may not be the initial location of the target if it has been moved.

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letobjectx(N,"shirt.cel") ; N contains the current X for the specified cel
;@letobjectx(N,#3) ; N contains the current X for object #3

 

 Example: 

;[Cel Section]
#4 rightshoe.gif : 0 ;
#4 leftshoe.gif : 0 ;
#4 sailormoon.gif : 0 ;
 
;[FKiSS Section]
;@EventHandler
 
;@press(#4) ;
;@ letinitx(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - initial X ",N) ;
;@ move(#4,10,0) ;
;@ letobjectx(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - current X ",N) ;
;@ letinitx(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - initial X ",N) ;
