random()

Syntax:     random (result,operand1,operand2)

This command computes a pseudo-random integer number. The result variable is is assigned a random number between operand1 and operand2.  Operand 2 should be greater than operand 1. The random number can equal operand 1 but will be less than operand 2. The result parameter must be a variable. Operand1 and operand2 can be numeric literal values or variables that represent numeric values.

FKiSS 3 and FKiSS 4 only support Integer values that range from -2147483648 to 2147483647, inclusive.  UltraKiss FKiSS 5 provides support for floating point numbers and long integers that can range from -9223372036854775808 to 9223372036854775807, inclusive. With FKiSS5 implicit conversion between variables is automatically performed and all calculations maintain the appropriate precision. 

Related actions: add(), sub(), mul(), div(), math(), sqrt()

 

Compatibility:

FKiSS 3 random(X,A,B)   A <= X < B 
FKiSS 5 random(MyVariable,A,3.145) or random(X,A,999999999999) int(A) <= X < int(B)

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

Example Code (FKiSS3): 

;@let(A,1) let(B,2) let(C,3)                         ; 
;@random(X,A,C)                                      ; X = 1 or 2

Example Code (FKiSS5): 

;@let(Apple,1) let(Pear,2) let(PI,3.1415927)         ; 
;@random(TwoPie,Apple,PI)                            ; TwoPie = 1 or 2
