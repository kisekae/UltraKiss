concat()

Syntax:     concat (variable,operand1,operand2,operand3,...)

The concat() statement concatenates a list of operands and returns the string result in the specified variable. Operands may be string literals, character literals, numbers, or variables that contain strings, characters, or numbers. All operand vaules are converted to string representation. The concat() statement allows for an unlimited number of operands.

Related actions: substr(), replacestr(), strlen(), indexof()

 

Compatibility:

FKiSS 5   concat (S,"The"," quick brown ",'\n',"fox")  S becomes "The quick brown \nfox"

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)
