letfix()

Syntax:     letfix (variable,target)

The letfix() statement returns the fix value associated with the cel or object in the specified variable.

Object fix count locks objects on the screen so they are unmovable. Fix count is initially established by lock values specified for image cels in the KiSS configuration file, however fix count can change when the user interacts with an object. The object fix count is reduced by one for each mouse interaction on the object. Note that fix count can only be reduced by user mouse action if the object fix value is less than the fixed lock value.

It is the object that is fixed and the fix count for an object group is usually defined as the maximum value of all lock values for all cels in the object, although some viewers may define this differently.

The target can be a cel name or an object group specifier. FKiSS 3 allowed only object groups for the letfix() statement, FKiSS 5 allows cel names.

 

Related events: unfix()

Related actions: setfix()

 

 

Compatibility:

FKiSS 5   letfix(N,"shirt.cel")  Variable N is set to the declared lock value for "shirt.cel"

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letfix(N,"shirt.cel") ; N contains the fix value for the specified cel
;@letfix(N,#3) ; N contains the largest fix value for all cels in the object #3
