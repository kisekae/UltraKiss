setpal() 

Syntax:     setpal (target,variable)

This action sets the multipalette number, also known as a palette group in a cel palette file (KCF file) to be used for coloring a cel, group, or celgroup. Note that multipalette file numbers begin from 0.

KiSS palette files can contain up to 10 different color palettes that are either 16 color or 256 colors in size. The optional palette group specification is an UltraKiss extension in the CNF file to identify the fixed palette group of colors that will be applied to the image. This fixed palette group will not change as color sets change and is related to the setpal() and letpal() FKiSS action commands. If the new palette file does not have enough multipalettes or is not compatible then no color change will occur to the cel image.

For group objects or cel groups the multipalette change is applied to all image cels within the object.

The target can be a cel name or an object group specifier or a cel group specifier. The variable must evaluate to be an integer value.

Related actions: letpal(), letkcf(), setkcf()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ setpal("256-col1.cel",N)     ;
;@ notify("This cel now references multipalette ",N," in the cel palette file ")    ;
