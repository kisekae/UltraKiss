letsibling()

Syntax:     letsibling (variable,object)

The letsibling() statement returns the next object in the attachment chain for the specified object. The children of the parent of the specified object are identified. Each child is an object with a unique object number. The next object number greater than the specified object found in the set of children of the parent is returned in the variable.

It is possible to iterate through all siblings by repeatedly issuing letsibling() commands on each returned object.

If no parent object exists or no remaining sibling objects exist the variable is set to -1.

The object must be an object group specifier or a variable containing an object group identifier.

 

Related actions: attach() detach() letparent() letchild()

 

 

Compatibility:

FKiSS 5   letsibling(N,#3)  Variable N is set to the only parent of group object 3.

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letsibling(N,"shirt.cel") ; N contains the current height for the specified cel
;@letsibling(N,#3) ; N contains the current height for object #3
