letmaxcolor() 

Syntax:     letmaxcolor (variable)

This action returns the maximum number of color sets currently defined in the KiSS set. Note that color numbers begin from 0. Note that with FKiSS 5 an unlimited number of palette groups or color sets can be defined for a KiSS KCF file.

Related actions: col(), changecol(), letmaxpage()

 

UltraKiss Options:

MaxColorSet specifies the maximum number of color sets displayed on the Ultrakiss toolbar. This is not the maximum number of color sets that can be defined. default value "10" (Compatibility option)

 

 Example: 

;@letmaxcolor(N)    ;
;@notify("This set has ",N," color sets")    ;
