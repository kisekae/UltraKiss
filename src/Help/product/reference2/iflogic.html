ifequal()

Syntax:     ifequal (operand,operand)

The ifequal() statement begin a conditional if-else-end block. The actions under it are carried out if the first operand has the same value as the second. Each operand can be a literal value or a variable. With FKiSS5 all if-statement blocks should be terminated with an endif() statement and all if-statement logic can be nested

Related actions: ifnotequal(), ifgreaterthan(), ifflessthan(), else(), endif(), elseifequal()

;@ifequal(A,1) map(#1) endif()                       ; 

ifgreaterthan()

Syntax:     ifgreaterthan (operand,operand)

The ifgreaterthan() statement begin a conditional if-else-end block.The actions under it are carried out if the first operand has a higher value than the second. Each operand can be a literal value or a variable. With FKiSS5 all if-statement blocks should be terminated with an endif() statement and all if-statement logic can be nested

Related actions: ifequal(), ifnotequal(), iflessthan(), else(), endif(), elseifgreaterthan()

;@ifgreaterthan(A,B) map(#1) endif()                 ;

else()

Syntax:     else ()

The else() statement is an optional part of the if-else-end block: the actions under it are carried out if the condition of the "if" statement is false. The else() statement requires a preceding "if" statement and a terminating endif() statement. With FKiSS5 all if-statement blocks should be terminated with an endif() statement and all if-statement logic can be nested

Related actions: elseifequal(), elseifnotequal(), elseifgreaterthan(), elseiflessthan(), endif()

;@ifequal(1,B) map(#1) else unmap(#1) endif()         ; 

elseifequal()

Syntax:     elseifequal (operand,operand)

This is an extended else() statement introduced with FKiSS 4; the actions under it are carried out if the first and second operand values are the same. Each operand can be a literal value or a variable. Requires an "if" statement and a terminating endif() statement. With FKiSS5 all if-statement blocks should be terminated with an endif() statement and all if-statement logic can be nested

Related actions: ifequal(), else(), endif(), elseifnotequal(), elseifgreaterthan(), elseiflessthan()

;@ifequal(A,B) map(#1)                                ; 
;@elseifequal(B,C) map(#2)                            ; 
;@endif()                                             ;

endif()

Syntax:     endif ()

The endif() statement concludes an if-else-end block or an if-end block. An endif() statement requires a preceeding if-statement. Note that early viewers for FKiSS 3 did not enforce the requirement that all if-statements be terminated with an endif(). This oversight precluded the nesting of if-statements and restricted the utility of FKiSS as a functional language. Endif() statements are required with all variants of the else() statement and are now enforced for all if-statements with FKiSS 5.

Related actions: ifequal(), ifnotequal(), ifgreaterthan(), iflessthan()

;@ifequal(A,B) map(#1) endif()                        ; 

 

Compatibility:

FKiSS 3 ifequal(A,B) integer comparisons, single letter variable names, optional endif()
FKiSS 4  ifnotequal(3,C) integer comparisons, extended else statements, long variable names, optional endif()
FKiSS 5 ifgreaterthan(A,"abc") ... endif() integer, string, floating point comparisons, extended else statements, long variable names, nested if statements, mandatory endif()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)
AutoEndIf automatically generate endif() statements default value "false" (Compatibility option)
StrictSyntax enforce strict block syntax checks default value "true" (Compatibility option)

 

Example Code (FKiSS3): 

;@let(A,1) let(B,2) let(C,3)                         ; 
;@ifequal(A,1) map(#1)                               ; unterminated "if"
;@ifnotequal(2,B) map(#1) endif()                    ; terminated "if"
;@ifgreaterthan(A,B) map(#1) else() map(#2) endif()  ; if-else-endif

Example Code (FKiSS4): 

;@let(ABC,1) let(BBC,2) let(CBC,3)                   ; 
;@ifequal(ABC,1) map(#1)                             ; unterminated "if"
;@ifnotequal(2,BBC) map(#1) endif()                  ; terminated "if"
;@ifgreaterthan(ABC,B)                               ;
;@ map(#1)                                           ;
;@elseifnotequal(BBC,CBC)                            ;
;@ map(#2)                                           ;
;@elseiflessthan(ABC,5)                              ;
;@ map(#3)                                           ;
;@else()                                             ;
;@ debug("Ooops")                                    ;
;@endif()                                            ; if-else-endif

Example Code (FKiSS5): 

;@let(ABC,1) let(BBC,"2") let(CBC,3.14)              ; 
;@ifequal("X",BBC) map(#1) endif()                   ; terminated "if" mandatory
;@ifgreaterthan(ABC,BBC)                             ; nested "if" (level 1)
;@ iflessthan(BBC,CBC)                               ; nested "if" (level 2)
;@  map(#1)                                          ;
;@ elseifequal(ABC,1)                                ;
;@  ifnotequal(ABC,BBC)                              ; nested "if"  (level 3)
;@   map(#2)                                         ;
;@  endif()                                          ; endif (level 3)
;@ else()                                            ;
;@  debug("Ooops")                                   ;
;@ endif()                                           ; endif (level 2)
;@endif()                                            ; endif (level 1)
