elseifgreaterthan()

Syntax:     elseifgreaterthan (operand,operand)

This is an extended else() statement introduced with FKiSS 4; the actions under it are carried out if the first operand value is higher than the second. Each operand can be a literal value or a variable. Requires an "if" statement and a terminating endif() statement. With FKiSS5 all if-statement blocks should be terminated with an endif() statement and all if-statement logic can be nested

Related actions: ifgreaterthan(), else(), endif(), elseifequal(), elseifnotequal(), elseiflessthan()

Nested if-statement are allowed.

 

 

Compatibility:

FKiSS 3 ifequal(A,B) integer comparisons, single letter variable names, optional endif()
FKiSS 4  ifnotequal(3,C) integer comparisons, extended else statements, long variable names, optional endif()
FKiSS 5 ifgreaterthan(A,"abc") ... endif() integer, string, floating point comparisons, extended else statements, long variable names, nested if statements, mandatory endif()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)
AutoEndIf automatically generate endif() statements default value "false" (Compatibility option)
StrictSyntax enforce strict block syntax checks default value "true" (Compatibility option)

 

Example Code (FKiSS3): 

;@let(A,1) let(B,2) let(C,3)                         ; 
;@ifequal(A,1) map(#1)                               ; unterminated "if"
;@ifnotequal(2,B) map(#1) endif()                    ; terminated "if"
;@ifgreaterthan(A,B) map(#1) else() map(#2) endif()  ; if-else-endif

Example Code (FKiSS4): 

;@let(ABC,1) let(BBC,2) let(CBC,3)                   ; 
;@ifequal(ABC,1) map(#1)                             ; unterminated "if"
;@ifnotequal(2,BBC) map(#1) endif()                  ; terminated "if"
;@ifgreaterthan(ABC,B)                               ;
;@ map(#1)                                           ;
;@elseifnotequal(BBC,CBC)                            ;
;@ map(#2)                                           ;
;@elseiflessthan(ABC,5)                              ;
;@ map(#3)                                           ;
;@else()                                             ;
;@ debug("Ooops")                                    ;
;@endif()                                            ; if-else-endif

Example Code (FKiSS5): 

;@let(ABC,1) let(BBC,"2") let(CBC,3.14)              ; 
;@ifequal("X",BBC) map(#1) endif()                   ; terminated "if" mandatory
;@ifgreaterthan(ABC,BBC)                             ; nested "if" (level 1)
;@ iflessthan(BBC,CBC)                               ; nested "if" (level 2)
;@  map(#1)                                          ;
;@ elseifequal(ABC,1)                                ;
;@  ifnotequal(ABC,BBC)                              ; nested "if"  (level 3)
;@   map(#2)                                         ;
;@  endif()                                          ; endif (level 3)
;@ else()                                            ;
;@  debug("Ooops")                                   ;
;@ endif()                                           ; endif (level 2)
;@endif()                                            ; endif (level 1)
