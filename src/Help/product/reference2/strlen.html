strlen()

Syntax:     strlen (variable,string)

The strlen() statement returns the number of characters in a string. If the string variable has not been previously defined the value returned is 0. An empty string also returns a 0 value.

Related actions: concat(), substr(), replacestr(), indexof()

 

Compatibility:

FKiSS 5   strlen (N,"The quick brown fox")  N becomes 19

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@strlen(N,"Hello") ; Variable N is set to 5
;@strlen(N,"") ; Variable N is set to 0
