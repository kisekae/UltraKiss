letkeystring()

Syntax:     letkeystring(variable)

The letkeystring() statement is a convenience function to return the last full line of text typed on the keyboard. Text lines are terminated when an 'enter' key is pressed. The letkeystring() statement would typically be used in a keyrelease("Enter") event to identify the end of line key entry.

The returned values are strings that exclude the terminating line feed character.

Some viewers may only recognize keyboard entry while a key is actually being pressed. This can lead to viewer compatibility problems as the letkeychar() statement may only function within keypress() events.

 

Related events: keypress(), keyrelease(), keytype()
Related actions: letkeymap(), letkeychar(), letkeycode(), letkey(), letkeymodifier()

 

UltraKiss Options:

KeyCase Enable keyboard upper/lower case key entry default value "false" (FKiSS option)
RetainKey Remember last key pressed default value "true" (Compatibility option)

 

 Example: 

;@keypress("Enter")
;@ letkeystring(S) ;
;@ notify(S)
