lettransparent()

Syntax:     lettransparent (variable,target)

The lettransparent() statement returns in the specified variable the current transparency value of the target cel, object group, or cel group. The transparency is an integer number from 0 to 255 where 0 represents a fully opague target and 255 represents a fully transparent target.

Cels maintain their own individual transparency value. The lettransparent() statement when the target is a cel returns the individual cel transparency.

Object groups and cel groups maintain one transparency value for the object. Relative transparency changes to an object adjust the object transparency and the relative adjustment is applied to each cel in the object. The lettransparent() statement when the target is an object group returns the object transparency which can be different from an individual cel transparency.

If the UltraKiss option to maintain group transparency values is not set, the lettransparent() statement when the target is an object group or a cel group returns the actual group cel transparency value if all cels in the group have the same transparency, otherwise the returned value is -1.

If the target cannot be identified the variable is not initialized or changed.

 

Related actions: transparent()

Compatibility:

FKiSS 1 allows cel and object group targets lettransparent(N,#12) The value for N is either -1 or bounded from 0 to 255
FKiSS 4 allows cel group targets lettransparent(N,!Clothing) The value for N is either -1 or bounded from 0 to 255
FKiSS 5 allows unbounded transparencies and variable targets let(Target,"!Clothing")
lettransparent(N,Target)		 The value for N is unbounded

 

UltraKiss Options:

allambiguous applies transparency adjustments to all ambiguously named cels default value "true" (Compatibility option)
transparentgroup lets object groups and cel groups maintain their own transparency adjustment. The lettransparent() statement returns this value and does not return -1 if the cels in the group have different transparencies. default value "true" (Compatibility option)
contextmap uses the current page context to disambiguate cel choices where ambiguous cels exist. default value "true" (Compatibility option)

 

Example: 

;@lettransparent(N,"shirt.cel") ; Variable N is set to the individual cel transparency
;@lettransparent(N,#12) ; Variable N is set to the object group transparency
;@lettransparent(N,!Clothing) ;
;@lettransparent(N,Target) ; The variable 'Target' must identify a cel, object, or cel group
