replacestr()

Syntax:     replacestr (variable,source,pattern,replacement[,count,case])

The replacestr() statement replaces a string pattern in a source string with a replacement string and returns the result in the specified variable. The optional integer count argument can be used to specify a maximum replacement count for the substitutions. If the count is not specified only the first occurance of the pattern in the source string is changed. If the count value is negative all occurances of the pattern in the source string are changed.

Pattern comparison in the source string is case sensitive if the optional case sensitive option is set to a non-zero value, otherwise the comparison is not case sensitive. The default is case sensitive comparisons.

Related actions: concat(), substr(), strlen(), indexof()

 

Compatibility:

FKiSS 5   replacestr (S,"The quick brown fox","quick","slow" )  S becomes "The slow brown fox"

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@replacestr(A,"John is smart","John",Susan") ; Variable A is set to "Susan is smart"
;@replacesrt(A,"xxxbxxx","x","a",-1) ; Variable A is set to "aaabaaa"
;@replacesrt(A,"xxxbxxx","x","A",-1) ; Variable A is set to "xxxbxxx"
;@replacestr(S,"x y z"," ","",-1) ; Variable S is set to "xyz"
