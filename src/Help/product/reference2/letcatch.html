letcatch() 

Syntax:     letcatch (variable[,group])

This action if specified as letcatch(variable) stores the group object number of the last pressed cel in the variable. If the command is specified as letcatch(variable,group) then the group object is tested for a collision and if any cel collided with another then the group number of the colling group is returned in the variable.

This command is typically used in a press event or other event to identify the currently selected group object.

The group parameter of the command must be a group object identifier.

Related actions: letmapped(), letcel()

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@ letcatch(GroupName)     ;
;@ notify("The group object last selected is ",GroupName)    ;
