sqrt()

Syntax:     sqrt (result,operand1)

This command computes the integer square root of a number. The result variable is is assigned the truncated value of the square root of a number..  The result must be a variable. Operand1 can be numeric literal values or variables that represent numeric values. 

result = floor(sqrt(operand1)). 

UltraKiss FKiSS 5 provides support for floating point numbers and long integers that can range from -9223372036854775808 to 9223372036854775807, inclusive. With FKiSS5 implicit conversion between variables is automatically performed and all calculations maintain the appropriate precision. 

Numeric overflow may occur if the resulting difference cannot be accurately represented by the computer. Numeric overflow will occur on computation of the square root of a negative number.

Related actions: add(), sub(), mul(), div(), mod(), math(), random()

 

Compatibility:

FKiSS 5 sqrt(MyVariable,3.145) or sqrt(X,999999999999) integer, string, floating point subtraction, long variable names

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

Example Code (FKiSS5): 

;@let(Apple,4) let(Pear,8) let(PI,3.1415927)         ; 
;@sqrt(X,Apple)                                      ; X = 2
;@sqrt(X,Pear)                                       ; X = 2
;@sqrt(X,PI)                                         ; X = 1
