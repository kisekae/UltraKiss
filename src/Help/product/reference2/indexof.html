indexof()

Syntax:     indexof (variable,string,substring[,case])

The indexof() statement returns the character index of the start of the substring within the string. Character indexing in a string begins with 0, so if the substring is at the start of the string the result variable will be set to 0. If the substring does not exist in the string the result variable will be set to -1.

Pattern comparison in the source string is case sensitive if the optional case sensitive option is set to a non-zero value, otherwise the comparison is not case sensitive. The default is case sensitive comparisons.

Related actions: concat(), substr(), replacestr(), strlen()

;@indexof(N,"OneTwoThree","Two") ; Variable N is set to 3
;@indexof(N,"OneTwoThree","Four") ; Variable N is set to -1
;@indexof(N,"OneTwoThree","TWO",0) ; Variable N is set to 3
