letchild()

Syntax:     letchild (variable,object)

The letchild() statement returns the lowest numbered object attached in the object's children.

As objects are numbered by the order in which they appear in the CNF, the lowest numbered child object is the first child.

If no children exist the variable is set to -1.

The object must be an object group specifier or a variable containing an object group identifier.

 

Related actions: attach() detach() letsiblingx() letparent()

 

 

Compatibility:

FKiSS 5   letchild(N,#3)  Variable N is set to the first child of group object 3.

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letchild(N,"shirt.cel") ; N contains the current height for the specified cel
;@letchild(N,#3) ; N contains the current height for object #3
