letwidth()

Syntax:     letwidth (variable,target)

The letwidth() statement returns the visible width of an object in pixels in the specified variable.

The visible width is context sensitive to the page that is currently visible. The same object on different page can have a different width.

The target can be an object group specifier.

 

Related actions: letheight()

 

 

UltraKiss Options:

VariableCase variable names are case sensitive default value "false" (Compatibility option)

 

 Example: 

;@letwidth(N,#3) ; N contains the visible width for object #3

 

 Example: 

;[Cel Section]
#4 rightshoe.gif : 0 ;
#4 leftshoe.gif : 0 ;
#4 sailormoon.gif : 0 ;
 
;[FKiSS Section]
;@EventHandler
 
;@press(#4) ;
;@ letwidth(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - height ",N) ; Answer is 109
;@ unmap("sailormoon.gif") ;
;@ letwidth(N,#4) ;
;@ notify("press Group 4 sailormoon.gif - height ",N) ; Answer is 85
