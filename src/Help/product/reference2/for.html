for() 

Syntax:     for (control,start,end[,increment])

This statement specifies the beginning of an iteration loop. The 'for' statement control parameter must be a variable. This variable is incremented or decremented by the increment amount on each iteration loop.

The control variable is initially set to the start value and the loop will terminate when the control variable exceeds the end value.

The start, end, and optional increment parameters can be integer constants or variables that contain integer values. The increment value, if not specified, defaults to 1 if the start value is less or equal to the end value and -if the end value is less than the start value.

The 'for' statement initializes the control variable to the start value on first entry. The 'for' statement executes any following code until the first matching 'next' statement for the same control variable is executed. This 'next' statement causes code execution to return to the currently preceeding 'for' statement with the same control variable.

On each subsequent 'for' statement entry the control variable is modified by the increment amount and the the resulting value is compared to the end value. If the increment is positive and the control variable exceeds the end value then the iteration loop is satisfied and code execution continues on the statement immediately following the 'next' statement that caused the 'for' statement re-entry.

It is possible to nest 'for' loops providing that each 'for' loop uses a unique control variable.

A 'for' loop can be terminated early by assigning the end value to the control variable.

 

Related actions: next(), while(), repeat()

;@ for(I,1,5)
;@  debug("I is ",I) 
;@  for(J,3,1)
