The Archive Manager Toolbar

The Archive Manager Toolbar is a control bar at the top of the screen that displays icons for common menu functions.  The toolbar icons and their functions are shown below. 

 

    New archive. New archive files are created from the New menu on the Archive Manager main program screen.  New archive files are initially empty and file elements can be added to the empty archive file with the Edit-Add command.

    Open archive. The Open command is used to open an existing archve file of an LZH, ZIP, or JAR type recognized by the Kisekae UltraKiss program. 

    Find element. The Find edit command is used to find elements of a particular type or name within the archive file element list.

    Add element to archive.  The Add command instructs the Archive Manager to add new file elements to the archive.  For detailed information see the Add Function document.

    Extract elements from archive. The Extract command instructs the Archive Manager to extract file elements from the archive into a file directory. For detailed information see the Extract Function document.

    View element. The View command instructs the Archive Manager to extract the first selected element from the archive element list and invoke the appropriate editor tool for the element.  Text files and KiSS configuration files are loaded into the Text Editor.  Image files are loaded into the Image Editor.  KiSS palette files are loaded into the Color Editor.  Sound files and movie files are loaded into the Media Player.
