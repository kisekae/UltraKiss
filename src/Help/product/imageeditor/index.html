Introduction

The Image Editor is a full featured graphical image editing program designed to manipulate graphical images used within KiSS data sets. The Image Editor provides many features for image manipulation, including scaling, rotation, clipping, affine transformations, and so on. This online Help document provides reference material for the various commands and options available within the Image Editor program.

The Image Editor tool is in development.  Not all functions are complete or operational. 

The Image Editor help files are incomplete.
