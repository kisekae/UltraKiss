Text Editor Help Topics

Welcome to the UltraKiss Text Editor documentation.  To open an online book, choose a book from the Contents page in the upper left of the Help Viewer. Click the book icon to expand the book topics. Then, click a topic to view it.

 

Introduction

The Text Editor is a full featured editor program designed to manipulate text files associated with KiSS data sets. The Text Editor supports standard and rich text files. This online Help document provides reference material for the various commands and options available within the Text Editor program.
