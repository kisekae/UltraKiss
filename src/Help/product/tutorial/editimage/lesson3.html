Image Tutorial

Lesson 3. Creating a Multi-Palette

 

  1. We are now going to create new different color variations to the doll set created in lesson 2.  If you are continuing from lesson 2 you can skip to step 3.  Otherwise, ensure that UltraKiss is showing the main logo screen.  Close any existing data set. This lesson requires editing of the KiSS set so ensure that the Edit Enable toggle button on the toolbar is set.

     

  2. Use File-Open  to load the configuration file CNF2.cnf from the Tutorial/Edit Image directory or the directory you used in the previous lesson.  This will load the doll set that you created in lesson 2.

     

  3. Switch to page 1 of the set with View-Page Set 1 or press the page 1 toolbar button. You should see two dolls, one with a blue shirt and one with a green shirt.  If this is the first time you have run this tutorial the background will be a very dark color, almost black.  We would like to change this to a light color as the dolls are meant to show on a light background.

     

  4. Right-click on top of the palette 0 toolbar button, or use View-Active Objects and select palette 0 from the palettes list.  Click on the Edit Palette button to invoke the color editor.

     

  5. Color index 0, the first box in the palette panel, should be selected and have a red outline.  In the Color Chooser panel above, click on a light yellow color swatch.  Color index 0 will change to match this color. Use File-Exit and answer Yes to update the active set.   

     

  6. Next, we will create a series of color variants for the doll clothes.  To do this we will create a palette file with many different palette groups.  Each color variant will be a different palette within the palette file.  Right-click with your mouse positioned over one of the skirts.  Select Image Context-Edit Palette.  This will invoke the Color Editor for the skirt.

     

  7. Use Edit-Select All to select all the colors in the palette.  Then, use Edit-Copy or Ctrl-c to copy all these colors to the clipboard.     

     

  8. Next, use Edit-Add Palette Group to create a new palette group within the file. This new palette will be all black.  Use Edit-Select Copy to re-select all the color indexes in the palette that were originally copied in Step 7, and then use Edit-Paste or Ctrl-v to paste the copied colors into the new palette.    

     

  9. To be precise we will explicitly search and select a light blue color to become our active color. Select the pointer tool in the Color Editor image preview panel and drag a small box around the skirt image being previewed.  When you release the mouse the preview image will be magnified so that you can see many of the different pixels. Hold the Ctrl key down and use the eyedropper tool to select a light blue color in the skirt.

    This will set the active color for relative color changes. This color is shown in the toolbar. If you forget to hold the Ctrl key down you will lose all the other color selections in the palette panel. If this happens, use Edit-Undo to cancel your selection and try again.

     

  10. Finally, choose a new color for the skirt from the color swatch in the Color Chooser and this color will be applied to the skirt.

    If you are not happy with the results, use Edit-Undo to undo the new color selection and ctrl-click with the eyedropper tool on a different color.  Repeat Step 10 until you are satisfied that all areas of the skirt are properly colored.

     

  11. You can repeat Steps 8 through 10 to create many different palette groups, each with a different skirt color.  When you are finished use File-Exit to exit the Color Editor, and answer Yes to apply your edits to the current set.

     

  12. You should now see multiple color palette buttons on the UltraKiss toolbar.  Click on these buttons or use View-Color Set to see your color variations.

     

  13. Repeat this lesson and create a new set of colors for the stockings.  In the next lesson we will apply transparency effects to the stockings.

     

  14. Use File-Save As to save your work.  Save the new KiSS set to the Tutorial/Edit Image directory or a directory of your choice.  Change the name to CNF3.cnf.  You can overwrite any existing element of the same name.
