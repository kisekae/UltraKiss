Visual Editing Tutorial

Lesson 1. Understanding How To Import Images

 

  1. This tutorial will show you how to bring new images into your KiSS set.  New images are the foundation of any Kisekae application. Kisekae UltraKiss is a visual development system.

    Ensure that UltraKiss is showing the main logo screen.  Close any existing set.

     

    This tutorial requires image files. If the necessary files cannot be found on your computer download the tutorial files using Help-Tutorial Files.

     

  2. Use File-New.  This will create a new KiSS set at size (800,600) with a black border and one empty page.

     

  3. Use File-Properties to change the new KiSS configuration (CNF) file properties.  The attributes section of the Properties dialog will show a playfield size of (800,600).  If it is not showing that size, press the small button beside this attribute and change the screen size to (800,600).  Exit the configuration Properties dialog by pressing the OK button.

     

  4. The first page of our set will be a menu page.  We will add two cel images that look like buttons to this page. We will use these buttons to switch to other pages in the set.  Each button will have a 'press' event attached to it.  The 'press' event will have the FKiSS code that we use to switch to a new page.

    On the main UltraKiss screen use Edit-Import Image to New Object.  Select the button1.cel file from the Tutorial/FKiSS directory.  This is a truecolor or Cherry Kiss cel so you will not see a prompt for an associated palette file.  The image will be loaded and positioned at the upper left corner of the playfield.

    Drag the button to the center of the screen.  Repeat this step to import button2.cel.  Position the second button below the first button.

     

  5. Note the difference between Edit-Import Image to Selection and Edit-Import Image to New Object. The first command imports images into the KiSS set that are intended to be grouped together into the same object. These images all have green marquee boxes. They are put together into one object with a new object number using the Edit-Regroup command.

    Images like the chest plate and the underwear that have front and back parts are imported as selections as these images are intended to be grouped together into one object.

    The Edit-Import Image to New Object is used when the image is intended to be placed into its own object. These images all have blue marquee boxes. New unique object numbers are assigned to these images when they are imported.

    It is not necessarily a problem if you imported images as objects and later decide you wanted them as image selections.

    To combine two or more objects into one new object, select the objects by right clicking and choosing Select while holding the CTRL key down. The CTRL key is used for multiple selections. You can also select multiple objects with the mouse by holding the ALT key down and drawing a selection box.

    When objects are selected use Edit-Ungroup to draw green marquee boxes around all image cels in the objects. When the images are all individually selected an Edit-New Group will combine all selected image cels into a new object. A new object number is assigned.

     

  6. Use File-Save As to save your work.  Save the new KiSS set to the Tutorial/FKiSS directory or a directory of your choice.  The configuration element name should be CNF1.cnf if you began this tutorial after first starting UltraKiss.  You can overwrite any existing element of the same name.
