Visual Editing Tutorial

Lesson 1. Understanding How To Select Images

 

  1. This tutorial will show you how to bring new images into your KiSS set and work with images in an existing KiSS set. Images are the foundation of any KiSS application. Kisekae UltraKiss is a visual application development system.

     

    Ensure that UltraKiss is showing the main logo screen.  Close any existing set. In this lesson you will learn the difference between an image cel and an object group. Cels are usually an image but can be other things like buttons or text boxes or movies. An object group is a collection of cels that together form a single object that can be manipulated as one unit.

     

    Ensure that UltraKiss is showing the main logo screen.  Close any existing set.

     

    This tutorial requires image files. If the necessary files cannot be found on your computer download the tutorial files using Help-Tutorial Files.

     

  2. Use File-New.  This will create a new KiSS set at size (800,600) with a black border and one empty page.

     

  3. Use File-Properties to change the new KiSS configuration (CNF) file properties.  The attributes section of the Properties dialog will show a playfield size of (800,600).  Press the small button beside this attribute and change the screen size to (600,600).  Exit the configuration Properties dialog by pressing the OK button.

     

  4. We are going to build a dynamic KiSS set that includes motion and animation. The first page of our set will be a menu page. Use Edit-Add Component and choose a button type component from the component selection dialog. We will add two button components to this page. We will also add two cel images to this page that look like buttons to compare and contrast image cel type buttons with component type buttons.

     

    The button component will be labelled button1 and will be positioned in the upper left corner of the playfield. It will have a green marquee box drawn around the image. This green marquee box indicates that the button is a cel image selected for editing.

     

    Drag the button to the center of the screen.

     

  5. Add another button. Use Edit-Add Component and choose a button type component from the component selection dialog. This button will be labelled button2 and will also have a green marquee box. This button has been imported into the same selection set as the first button.

     

    Drag this button to the center of the screen and position it below button 1.

     

    Both buttons should have green marquee boxes. Both buttons are part of the same selection set. If they do not both have green marquee boxes use Edit-Undo All and begin again from Step 4. Note, if you start again your buttons will now be named button3 and button4 as new component names are created each time a Component is added.

     

  6. Use Edit-Unselect All or right-click any where on the screen not over an object. The green marquee selection boxes will disappear as we are no longer selecting these cels for editing. The two button components are now combined as part of the same object number #1.

     

    Right-click on one of the component button images and choose Select from the pop-up menu. The two buttons will now be encased in a blue marquee box. The blue marquee box shows that this object contains both buttons and has been selected for editing.

     

    When objects or cels are selected for editing they can always be moved on the screen or otherwise manipulated regardless of any fix state or other restrictions that may apply within the KiSS set.

     

  7. Add another button. Use Edit-Add Component and choose a button type component from the component selection dialog. This button will be labelled button3 and will also have a green marquee box. The blue marquee box around the first two buttons will disappear. An object selection cannot be combined with an image selection. When you created a new button you created a new cel image selection.

     

    Drag this button to the center of the screen and position it below the other buttons.

     

    The first two buttons belong to object group #1. This third button, because it was not imported into an existing selection set, has created a new object in the configuration. This would be object group #2.

     

    Right-click anywhere on the screen not on an object to remove any current edit selection. Pick up the various button components with the mouse and move them on the screen. Observe that one object has two button components that move together and the other object has only one.

     

  8. We can play with these object groups. Select the object group that has two button components by right-clicking on one of the buttons and choosing Select from the pop-up menu. Both buttons will now be encased with a blue marquee box. This means that the object group is selected.

     

    Now, select the object group that has only one button components by right-clicking on this button and choosing Select from the pop-up menu. This button will now be encased with a blue marquee box. The other buttons will no longer be marked. This selection of the single button object has changed our object selection set.

     

    To make multiple selections of objects use the CTRL key. Hold the CTRL key down and right click while over one of the unselected buttons in object #1. Choose Select from the pop-up menu. This object group will be selected and now all buttons will now show a blue marquee box. Both object groups are now selected.

     

    Even though the objects are both selected each object group will still move independently. Use your mouse and move each object. To move all selected objects concurrently use the SHIFT key. Hold the SHIFT key down and drag. Observe the result.

     

  9. Use Edit-Unselect All or right-click somewhere on the screen not over an object to remove all selections. Now, position your mouse near the upper left of the playfield and use your mouse to draw a selection box around the buttons. Hold the ALT key key down as you drag your mouse. Select all the buttons. This is another way to perform a multiple selection.

     

    Both object groups are now selected but they are both independent objects as discussed in the previous step. To combine these objects into one object group use Edit-New Group. This will create a new group object with a new object number. The marquee box will flash. You can now pick up and move the new object with your mouse. All three buttons will move concurrently.

     

  10. Use View-Active Configuration or press F11 to open the Text Editor tool to view the CNF configuration element that has been created for this new KiSS set. Editing this text document is another way to edit our configuration and update our KiSS set.

     

    Select the first line in the [Cel Section] of the file. This line should refer to object #3 and button6.button. You will observe that various attributes are specified for this button after the last semicolon on the line. Another lesson will show how these attributes can be set.

     

    You may observe that the component name for the third button has changed from button3.button to button6.button. This is a side effect of the Edit-New Group command. Component names are internally generated when new components are created. For now, select the line containing this component name and press the Del key on your keyboard to delete this line.

     

    Use the File-Exit command to exit the text editor and apply the changes to the active set. The set will reload and the third button you created in Step 7 will be gone.

     

  11. We will now add two cel images that look like buttons to this page. On the main UltraKiss screen use Edit-Import Image to New Object.  Select the button1.cel file from the Tutorial/Visual Editing directory.  This is a truecolor or Cherry Kiss cel so you will not see a prompt for an associated palette file.  The image will be loaded as a new object and positioned at the upper left corner of the playfield. It will have a blue marquee box as it was imported as an object.

     

    Drag the button to the center of the screen and position it beside the other button components. 

     

  12. Now, use Edit-Import Image to Selection to import another image as a cel.  You may find that you cannot use the Import to Selection command as it is disabled. The previous selection imported an image as an object and it has a blue marquee box. Object selection sets cannot be combined with image selection sets.

     

    Use Edit-Unselect All or right-click somewhere on the screen not over an object to remove all selections. Now, repeat the command to Edit-Import Image to Selection.

     

    Select the button2.cel file from the Tutorial/Visual Editing directory.  This is another truecolor or Cherry Kiss cel so you will not see a prompt for an associated palette file. This image cel as it has been imported as a selection will be encased in a green marquee box.

     

    Position the second button below the first button.

     

  13. Note the difference between Edit-Import Image to Selection and Edit-Import Image to New Object. The first command imports images into the KiSS set that are intended to be grouped together into the same object. These images all have green marquee boxes. They are put together into one object with a new object number using the Edit-Regroup command.

     

    The Edit-Import Image to New Object is used when the image is intended to be placed into its own object. These images all have blue marquee boxes. New unique object numbers are assigned to these images when they are imported.

     

  14. We will now explore selecting cels. Use Edit-Unselect All or right-click somewhere on the screen not over an object to remove all selections.

     

    Select only the two image cels recently imported with the Press Me text. Select the objects by right clicking and choosing Select while holding the CTRL key down. You can also select multiple objects with the mouse by holding the ALT key down and drawing a selection box. Draw a box only around the two image cels recently imported. The objects when selected will both have blue marquee boxes.

     

    When the objects are selected use Edit-Ungroup to draw green marquee boxes around all image cels in all the objects. When images are all individually selected an Edit-Regoup will restore the selected image cels back into their original object.

     

    Although we have two images selected with green marquee boxes they are not part of the same object.

     

    For example, with the objects both ungrouped and showing green marquee boxes for the cels, use your mouse and drag one of the cels off to the side. Perform an Edit-Regroup and observe that the two cels are restored as individual objects in their new position. Both buttons should now have blue marquee boxes. Further, if you pick up and drag these two button cels with your mouse both cels move independently.

     

    Ungrouping and regrouping cels in this way lets us manipulate object cels within the object.

     

  15. When the images are all ungrouped an Edit-New Group will combine all selected image cels into one new object. A new object number is assigned. Combining cels in this way is identical to combining group objects as we did previously with the component buttons.

     

    With both button cels with the Press Me descriptions selected as objects with blue marquee boxes, use Edit-Ungroup once again to draw green marquee boxes around the objects. This time use Edit-New Group to combine the cels into one object. The blue marquee box will now show the boundary of the combined cels.

     

  16. Once again use Edit-Ungroup to convert the object to selected cels or right-click on one of the selected buttons and choose from the pop-up menu. This time right click on the cel image that you dragged to the side in the previous step and choose Select Cel from the pop-up menu. A yellow marquee box will be drawn around this cel image. A yellow marquee box indicates that this is a selected cel from within the set of cels selected in the object. Selected cels like this are used for copy and paste operations that will be explored in the following lesson.

     

    Use your mouse to drag this selected cel and position it so that one button is better aligned above the other button. Use Edit-Regroup to restore the object group with the repositioned cel.

     

  17. Use File-Save As to save your work.  Save the new KiSS set to the Tutorial/Visual Editing directory or a directory of your choice.  The configuration element name should be CNF1.cnf if you began this tutorial after first starting UltraKiss.  You can overwrite any existing element of the same name.
