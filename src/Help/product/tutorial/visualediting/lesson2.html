Visual Editing Tutorial

Lesson 2. Understanding Copy and Paste

 

  1. This tutorial will show you how to copy and paste image cels and objects within and across pages in your KiSS set.  You will learn about ambiguous cels, that are ambiguous because the same cel is used in more than one place on the page.

     

    Ensure that UltraKiss is showing the main logo screen.  Close any existing set.

     

    This tutorial requires image files. If the necessary files cannot be found on your computer download the tutorial files using Help-Tutorial Files.

     

  2. Use File-New.  This will create a new KiSS set at size (800,600) with a black border and one empty page.

     

  3. Use File-Save As to save your work.  Save the new KiSS set to the Tutorial/FKiSS directory or a directory of your choice.  The configuration element name should be CNF1.cnf if you began this tutorial after first starting UltraKiss.  You can overwrite any existing element of the same name.
