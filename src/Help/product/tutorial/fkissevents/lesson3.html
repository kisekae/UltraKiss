Event Tutorial

Lesson 3. Creating Snap-To Events

 

  1. We are now going to create collision or Snap-To events that help position clothes on a doll.  If you are continuing from lesson 2 you can skip to step 3.  Otherwise, ensure that UltraKiss is showing the main logo screen.  Close any existing data set.

     

  2. Use File-Open  to load the configuration file CNF2.cnf from the Tutorial/FKiSS directory or the directory you used in the previous lesson.  This will load the doll set that you created in lesson 2.

     

  3. Add a new page to the set.  Use Edit-Add Page to create page 3. 

  4. Our set has a black background but we are going to import an image that is meant to be shown on a light background. To fix this problem we will use a palette file for the set.  Use Edit-Import Palette and import the palette file 'background.kcf' from the Tutorial/FKiSS/Bearman directory.   The background color should change to a dark green and you will see two multipalette buttons appear on the toolbar..

     

  5. Next, use Edit-Import Image to New Object and import the file 'human.gif' from the Tutorial/FKiSS/Bearman directory.   This image will be automatically converted to a CEL file with an associated KCF palette. Drag the image to the center of the screen.

    Next, right-click on the page set 3 toolbar button or use View-Active Objects and expand the Page Sets line to select Page Set 3.  Press the little button beside the Palette Group attribute and change the default group from 0 to 1.  Page 3 will now be associated with palette group 1 in the background.kcf file and the background color will change to a light yellow.

     

  6. We will now import some clothes for our doll. Right-click anywhere on the background or use Edit-Unselect All to remove any current image selections. Use Edit-Import Image to Selection and import the files 'chestplate.gif' and 'chestplateback.gif'.  You can select both files concurrently by using the ctrl key for multiple selections.  The images will be imported to the upper left corner of the screen and should have green marquee selection boxes around them.  Green selection boxes indicate that the images are parts of a larger object group.

     

  7. Drag the chestplate back image to position it properly within the object.  If necessary, use View-Magnify to increase the magnification of the playfield for mor accurate positioning. Then, use Edit-Regroup to group the two chestplate images into one combined object.  The marquee box should turn blue and encase the complete object.

     

  8. Right-click anywhere on the background or use Edit-Unselect All to unselect the chestplate.  Drag the chestplate to position it properly on the human.     

     

  9. As you drag you should note that the back of the chestplate does not properly slide behind the human.  Use Edit-Adjust Layering to fix this. Select the chestplateback.gif image line and drag it below the human.gif image line.  This adjusts the drawing level so that the chestplate now appears correct on the human.

     

  10. We will now define a Snap-To event on the chestplate.  Right-click on the chestplate when it is properly positioned on the doll.  Select Image Context and choose the Event Wizard.

     

  11. Select the Snap To event in the Event Wizard.  A popup dialog will ask you to choose the parent object for the chestplate.  This is the human doll, so select the group object associated with human.gif.

    You will see the event code that is automatically generated for you.  Press the OK button and the snap-to event will be defined.  Test it.  Drag the chestplate completely off the doll and release it.  Pick it up again and place it anywhere on the doll, and it should snap to its proper position when you release the mouse.

     

  12. You can repeat Steps 5 through 10 to import the remaining pieces of clothing or armour for the human, an attach Snap-To events on these items. Note that the underwear also has front and back similar to the chestplate.  We leave this for you to do as an exercise.

     

  13. Use File-Save As to save your work.  Save the new KiSS set to the Tutorial/FKiSS directory or a directory of your choice.  Change the name to CNF3.cnf.  You can overwrite any existing element of the same name.
