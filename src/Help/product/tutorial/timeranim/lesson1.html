Animation Tutorial

Lesson 1. Create a New KiSS Set

 

  1. We will create a brand new KiSS set that shows how to animate or move objects on the screen.  Ensure that UltraKiss is showing the main logo screen.  Close any existing data set.

     

    This tutorial requires image files. If the necessary files cannot be found on your computer download the tutorial files using Help-Tutorial Files.

     

  2. Use File-New .  This will create a new KiSS set at size (448,320) with a black border and one empty page.

     

  3. Use View-Active Configuration to edit the KiSS set configuration (CNF) file.  In the [Screen Section] of the file change the screen size to (640,425) and  change the border colour to [4.  Exit the text editor and apply the changes.

     

  4. Use Edit-Import Image to Selection.  We are importing an image into a selection set rather than an object because more than one image is going to be grouped together into one object. Select the eagle.cel file from the Tutorial/Kisimi directory.  When you import CEL images you may also have to identify a KiSS palette file to be used with the image.  If prompted, choose the file sdkismi.kcf .  The image will be loaded and positioned near the upper left corner of the playfield.  Leave the image where it is.

    You will also see the background colour of the playfield change to a light brown.  This occurs because we just added the first palette file, sdkismi.kcf, to the configuration.  The playfield background colour is derived from this palette and in this case it is the first colour from the first palette.

     

  5. Use Edit-Import Image to Selection and import the smoke1.cel file from the Tutorial/Kisimi directory, and choose the sdkismi.kcf palette.  You will note that the smoke image is properly positioned behind the airplane.  This is because both the eagle and the smoke cels have image offset values set in their image files.  These offset values are displacements from the origin.

    If you inadvertently import an image to an object instead of a selection then the image will have a blue marque box around it. To correct this you can unselect everything (Edit-Unselect All) and then select all the required objects (Ctrl-right click and choose Select from the pop-up dialog), and then use Edit-Ungroup to convert all images to a selection set. Every image in a selection set should be drawn with a green marque box.

     

  6. Use Edit-Import Image to Selection and import the smoke2.cel file from Tutorial/Kisimi.

     

  7. Use Edit-New Group to group the three image cels into one object group.  A single blue selection box should now enclose the complete object.

    An Edit-New Group puts all the images in the selection set into a new group object. A new object number is assigned. This is required if you had inadvertently imported any of the smoke images to an object rather than the active selection set.

     

  8. Use View-Restart to reload the data set.  This will establish the new palette so that the border colours work properly.  When the data set is reloaded you should see the border colour change from black to a dark peach colour.  This is the fourth colour in the palette that we set in step 3.

     

  9. Try to pick up the airplane and drag it.  The airplane and the smoke should move as one object.

     

  10. Use File-Save As to save your work.  Save the new KiSS set to the Tutorial/Kisimi directory or a directory of your choice.  The configuration element name will be CNF1.cnf if you began this tutorial after first starting UltraKiss.  You can overwrite any existing element of the same name.

     

  11. Use File-Close to close the data set.  UltraKiss will now show the main logo screen.

     

  12. To check your work use File-Open to load the configuration file CNF1.cnf from your directory.
