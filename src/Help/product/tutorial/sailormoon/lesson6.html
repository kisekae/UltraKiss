Sailor Moon Tutorial

Lesson 6. Adding Sound

 

  1. We will now add sounds to our KiSS set.  It is often appropriate to add start-up and shut-down sounds.  If you are continuing on from lesson 5 you can skip to step 3. Otherwise, ensure that UltraKiss is showing the main logo screen.  Close any existing data set. If you want to ensure that your new KiSS set is created to be compatible with other viewers, press one of the viewer compatibility preset buttons on the toolbar.

     

  2. Use File-Open.  Load the configuration file CNF4.cnf from the Tutorial/Sailor Moon directory or the directory you used in lesson 5.  This will load the corrected Sailor Moon KiSS set created in the last exercise 5.

     

  3. Use Edit-Import Other-Import Audio to open a file selection dialog to add audio files into your KiSS set. Select the sound files "smc181-2.wav", "smc308-2.wav", "cat.wav", "catsound.wav", "angrycat.wav", and "catscrm.wav" from the Tutorial/Sailor Moon directory and import these files.

     

  4. Use View-Active Configuration to open the text editor for the current configuration file.  Position the cursor at the end of the file.

     

  5. Initial actions for a KiSS set are coded in a begin() event.  Terminating actions for a set are coded in an end() event.  We will use the audio files "smc181-2.wav" and "smc308-2.wav " for start-up and shut-down sounds.  Add the following lines to the end of the configuration file:

    ;@begin()
    ;@ sound("smc181-2.wav")

    ;@end()
    ;@ sound("smc308-2.wav")

    Note:  If you copy and paste these lines from the help documentation into your configuration file you must ensure that the semicolon and at-sign combination appears at the beginning of every line.  You can copy the help text by selecting the text and pressing Ctrl-C to copy the text to the clipboard. Use Ctrl-V to paste the lines into the text editor.


  6. We may as well add some cat sounds.  We will have the cat make a random sound when the mouse is clicked while on the cat.  Code a release() event for the cat as follows:

    ;@release(#9)
    ;@ random(A,0,9)
    ;@ iflessthan(A,7) sound("cat.wav")
    ;@ elseiflessthan(A,8) sound("catsound.wav")
    ;@ elseifequal(A,8) sound("angrycat.wav")
    ;@ elseifequal(A,9) sound("catscrm.wav")
    ;@ endif()
    This event generates a random number and uses this to select a cat sound.  Most of the time you should hear "cat.wav" but occasionally you may hear another sound.

     

  7. Use the File-Exit command in the text editor to apply your changes.  If you made a typing mistake you may see an error when the changes are applied.  If this happens, re-edit the configuration file to correct any mistakes.  Otherwise, the data set should reload and open to page 0.  You should hear the begin sound as the revised data set restarts. Test the changes.

     

  8. Use File-Save As to save your work.  Use a new CNF5.cnf file name.  This new configuration will contain the sound event code and the sound files should be saved to your directory.

     

  9. Restart the data set with the View-Restart command.  This will close and reload the current configuration.  You should hear the begin sound as the revised data set restarts.

     

  10. Press on the cat on page 1 and you should hear a random cat sound.

     

  11. Use File-Close to close the data set and you should hear the end sound.

    This completes the tutorial to create your first KiSS set.
