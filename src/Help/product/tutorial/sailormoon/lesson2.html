Sailor Moon Tutorial

Lesson 2. Adding Images to a KiSS Set

 

  1. We will now add the remaining sailor costume images to the Sailor Moon data set.  If you are continuing from lesson 1 you can skip to step 3.  Otherwise, ensure that UltraKiss is showing the main logo screen.  Close any existing data set. If you want to ensure that your new KiSS set is created to be compatible with other viewers, press one of the viewer compatibility preset buttons on the toolbar or use Tools-Options-Compatibility to set a compatibility preset. This lesson requires editing of the KiSS set so ensure that the Edit Enable toggle button on the toolbar is set.

     

  2. Use File-Open  to load the configuration file CNF1.cnf from the Tutorial/Sailor Moon directory or the directory you used in lesson 1.  This will load the Sailor Moon KiSS set that you created in lesson 1.

     

  3. Use Edit-Import Image to New Object.  Select the fuku.gif file from the Tutorial/Sailor Moon directory.  The image will be loaded and positioned at the upper left corner of the playfield. If any viewer preset compatibility option is set the image will be converted to a CEL file with an appropriate KCF. This conversion is performed if the Options-Options-Import setting to import images as CEL files is set.

     

  4. Drag the image and position it on the doll.

     

  5. Use Edit-Import Image to New Object and import the images leftglove.gif, rightglove,gif, and bow.gif.  You can import these images all at once if you hold the ctrl key down and make multiple selections.  Position these objects appropriately on the doll.

     

  6. You can use Edit-Unselect All to remove the selection boxes from the images and terminate your editing.  Or, simply right-click with your mouse pointer while it is not over an object to unselect all objects.

     

  7. Use View-Restart or press F8 to reload the set in UltraKiss and commit your changes.  Or, you can use Edit-Write Page to set the initial positions for the new imported objects after you have moved them to their final location.  Once the initial positions are set a View-Reset command will always restore the objects to these positions.

     

  8. As you play with your set you may find that some objects have been hidden behind other objects or do not overlay one another properly.  For example, the bow should appear in front of the fuku.  If the objects are not layered correctly you can adjust their layering order in the configuration file.

    Use the Edit-Adjust Layering command to change the object layering order.  Select the object of interest within the list and drag it to the proper position.  Objects at the top of the list appear in front of all lower objects.  If the list does not want to scroll down, ensure that your mouse pointer is near the lower left or lower right corner of the dialog window.  Release the mouse button to drop the object.  Use the OK button to apply your changes.
    You can use the Undo button in the layer adjustment dialog to undo your change if you make a mistake while adjusting the layers. If you have applied your changes and exited the layer dialog you can undo all the layering changes with the Edit-Undo Layer Adjustment function on the main menu.

     

  9. Use File-Save As to save your work.  This time change the name to CNF2.cnf .  The new configuration file and the new imported images will be saved in the same directory as the original CNF1.cnf file.  This will commit your changes and set the initial positions for all new imported objects to their current locations.

     

  10. Check your work.  You can pick up and move various objects as required.  The View-Reset command will restore the objects to their original positions.  If you need to correct any object initial location, move the object to its desired position and use the Edit-Write Page command to update the initial locations for all objects on the page.

     

  11. Use File-Save to save any object layering changes or initial location adjustments that you have made.
