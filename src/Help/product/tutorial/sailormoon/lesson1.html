Sailor Moon Tutorial

Lesson 1. Create a New KiSS Set

 

  1. We will create a brand new Sailor Moon KiSS set.  Ensure that UltraKiss is showing the main logo screen.  Close any existing set. If you want to ensure that your new KiSS set is created to be compatible with other viewers, press one of the viewer compatibility preset buttons on the toolbar or use Options-Options-Compatibility to set a compatibility preset. This lesson requires editing of the KiSS set so ensure that the Edit Enable toggle button on the toolbar is set.

     

    This tutorial requires image files. If the necessary files cannot be found on your computer download the tutorial files using Help-Tutorial Files.

     

  2. Use File-New.  This will create a new KiSS set at size (800,600) with a black border and one empty page.

     

  3. Use File-Properties to change the new KiSS configuration (CNF) file properties.  The attributes section of the Properties dialog may show a playfield size of (800,600).  If it does press the small button beside this attribute and change the screen size to (600,400).  Next, press the small black button beside the border color attribute and select a white background (or other color) from the color swatch panel.  When you choose this color and exit the properties dialog the screen background will change to the selected color.  This is the outside background color for the screen. Exit the configuration Properties dialog by pressing the OK button.

     

  4. Use Edit-Import Image to New Object.  Select the sailormoon.gif file from the Tutorial/Sailor Moon directory.  The image will be loaded and positioned at the upper left corner of the playfield. If any viewer preset compatibility option is set the image will be converted to a CEL file with an appropriate KCF. This conversion is performed if the Options-Options-Import setting is set to Import all images as CEL files. The image when imported will also have a blue selection marquee box drawn around the image. This box shows that the image has been selected for editing.

    If the image was converted to a CEL file with a KCF file then the inside frame background color of the set will change. This background color is taken from the first color in the first palette defined in the configuration file.

     

  5. Drag the image to the center of the screen.

     

  6. Use Edit-Import Image to New Object.  Import the leftboot.gif file from the Tutorial/Sailor Moon directory.  Drag the image to position it properly on the doll.

     

  7. Use Edit-Import Image to New Object.  Import the rightboot.gif file from Tutorial/Sailor Moon.  Drag the image to position it properly on the doll.

     

  8. We will now put the two boots into one object.  Position the mouse pointer to the left and above the left boot, hold the alt key down and drag the mouse to the right and down, and select both the left and right boots.  For linux systems you may have to hold the shift key down rather than the alt key.  A red selection box will show the selection area as you drag.  When you release the mouse button both boots should be selected, with blue boundaries. If you happened to catch the sailormoon image along with the boots in the selection box just right click anywhere outside the box to remove the selection and try again to select only the boots.

     

  9. Use Edit-New Group to place both boots into the same object group.  A single blue selection box should now enclose both boots. 

     

  10. Use View-Restart or press F8 to reload the set in UltraKiss and commit your changes.  The initial positions will be set for all new objects imported into the set.  You should see the Sailor Moon doll and boots displayed correctly when the set is reloaded.

     

  11. Try to pick up the boots and drag them.  Both boots should move as one object.  You can use the View-Reset command or press F7 to restore all objects to their initial positions.

     

  12. Use File-Save As to save your work.  Save the new KiSS set to the Tutorial/Sailor Moon directory or a directory of your choice.  The configuration element name should be CNF1.cnf if you began this tutorial after first starting UltraKiss.  You can overwrite any existing element of the same name.  If you have selected one of the compatibility presets, seven files will be written.  There will be the three image files sailormoon.cel, leftboot.cel, rightboot.cel; the three associate kcf files; and the new data set configuration file CNF1.cnf.
