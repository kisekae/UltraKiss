Kisekae UltraKiss Installation

Kisekae UltraKiss is distributed as open source software from GitHub under the project Kisekae UltraKiss

The UltraKiss product has been developed as a platform independent application for Microsoft, Unix and Apple computer systems.

UltraKiss requires a Java run time environment.  Kisekae UltraKiss is supplied with a JRE version 17 system.

 

Effective with UltraKiss release 3.3b, Kisekae UltraKiss is distributed as a fully bundled application for Microsoft Windows systems.

Effective with UltraKiss release 3.6, Kisekae UltraKiss is distributed as a fully bundled application for Linux and Apple Mac OS systems.

Effective with UltraKiss release 4.0, Kisekae UltraKiss is distributed as a Webswing capable application to enable modern browser access.

Effective with UltraKiss release 4.2, Webswing is discontinued due to licensing issues. A replacement service is in development.

Effective with UltraKiss release 5.0, Kisekae UltraKiss is released as a client-server application that fully enables modern browser access.

No separate installation of Java software is required.

 

32-bit architecture systems can run UltraKiss directly from the kisekae.jar file using the supplied JRE version 8 system. See the download instructions on GitHub.
